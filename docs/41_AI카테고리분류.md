# AI 카테고리 분류 시스템

**날짜**: 2026-03-06

키워드 기반 분류의 신뢰도가 낮을 때 AI로 재분류하여 정확도를 높입니다.

---

## 문제점

### 케이스 1: 스포츠마케팅 오분류
```
제목: '2026 나주방문의 해' 스포츠마케팅으로 체류형 관광 전환 박차
키워드 분류: 스포츠 (❌ 잘못됨)
실제 카테고리: 사회 (지역 관광 정책)
```

### 케이스 2: 연예인 가짜뉴스 오분류
```
제목: SNS에 퍼진 '쿠엔틴 타란티노 이란 공격 사망설' 진실은?
키워드 분류: 스포츠? (❌ 잘못됨)
실제 카테고리: 사회 (가짜뉴스)
```

**핵심 문제**: 단순 키워드 매칭으로는 맥락을 이해할 수 없음

---

## 해결 방안: 하이브리드 분류

### 전략

```
1차: 키워드 기반 분류 (빠름)
   ↓
2차: 신뢰도 체크
   ↓ (낮음)
3차: AI 재분류 (정확함)
   ↓
4차: 최종 카테고리 결정
```

### AI 재분류 조건

다음 중 하나라도 해당하면 AI 재분류:

1. **키워드 점수 < 3** - 매칭된 키워드가 너무 적음
2. **다수결과 키워드 불일치** - 네이버 카테고리와 키워드 결과가 다름
3. **맥락 규칙 없고 점수 낮음** - 특별한 규칙 없이 점수만 낮음

### 비용 최적화

- 신뢰도 높을 때: 키워드만 사용 (무료, 빠름)
- 신뢰도 낮을 때: AI 사용 (Groq 무료)
- 예상 AI 호출: 하루 20-30회 (전체의 30-40%)

## 성능 영향

### Before (키워드만)
- 속도: 빠름 (~10ms)
- 정확도: 75-80%
- 비용: 무료

### After (하이브리드)
- 속도: 빠름 (~10ms) 또는 보통 (~500ms, AI 사용 시)
- 정확도: 90-95%
- 비용: 무료 (Groq)

### 예상 AI 호출

```
하루 이슈 생성: 50-100개
AI 재분류 필요: 20-30개 (30-40%)
Groq 무료 한도: 14,400회/일
사용률: 0.2% (여유로움)
```
