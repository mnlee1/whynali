# AI 대시보드 최종 정리

## 핵심 변경 사항

### 1. 네이버 API 제거
- AI 시스템이 아닌 데이터 수집 API는 대시보드에서 제외
- AI 시스템만 집중 표시

### 2. 실시간 사용 상태 감지
**기존 문제:**
- 이번 달에 호출이 1회라도 있으면 "사용 중"으로 표시
- Perplexity를 비활성화해도 계속 표시됨

**개선:**
- **오늘 호출이 있을 때만** Perplexity 카드 표시
- 실제로 사용 중일 때만 "사용 중" 배지 표시

```typescript
// 백엔드 로직
perplexity: perplexityCallsToday > 0 ? { ... } : null
```

### 3. 표시 정보

각 AI 카드에 표시되는 정보:

#### Groq AI (항상 표시)
- **오늘**: 호출 횟수 + 금액 ($0.00)
- **이번 달 누적**: 호출 횟수 + 금액 ($0.00)
- **활성 기능**: 커뮤니티 매칭, 중복 이슈 체크, 토론 주제 생성, 투표 생성
- **성공률**: 오류 감지용

#### Perplexity AI (오늘 사용 시에만 표시)
- **오늘**: 호출 횟수 + 금액
- **이번 달 누적**: 호출 횟수 + 금액
- **활성 기능**: 뉴스 그루핑, 이슈 후보 검증
- **토큰 사용량**: 입력/출력/전체 (K 단위)
- **성공률**: 오류 감지용

## Perplexity 활성화 시

### 환경변수 변경
```bash
ENABLE_AI_GROUPING=true
ENABLE_AI_GROUP_VALIDATION=true
```

### 재시작 후
- Perplexity API 호출 발생
- **다음 날 자동으로** Perplexity 카드가 나타남

## 동작 흐름

### 1. Perplexity 비활성화 → 활성화
1. 환경변수 변경 (`ENABLE_AI_GROUPING=true`)
2. 서버 재시작
3. Perplexity API 호출 시작
4. **다음 날** 대시보드에 자동으로 Perplexity 카드 표시

### 2. Perplexity 활성화 → 비활성화
1. 환경변수 변경 (`ENABLE_AI_GROUPING=false`)
2. 서버 재시작
3. Perplexity API 호출 중단
4. **다음 날** 대시보드에서 Perplexity 카드 자동으로 사라짐

## 장점

1. **정확한 상태 표시**: 오늘 사용 중인 AI만 표시
2. **자동 감지**: 환경변수만 바꾸면 대시보드가 자동으로 업데이트
3. **깔끔한 화면**: 사용하지 않는 AI는 완전히 숨김
4. **실시간 모니터링**: 오늘/월별 사용량, 성공률 확인

## 주의사항

- Perplexity 활성화 후 **다음 날부터** 대시보드에 표시됨
- 비활성화 후에도 **다음 날까지는** 표시될 수 있음
- 즉시 반영을 원하면 DB에서 해당 날짜 데이터 삭제 필요
