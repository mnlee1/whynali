### 3. 하이브리드 연결 로직 ✅

**파일:** `lib/linker/issue-news-linker.ts`

**프로세스:**

```
1차: 키워드 필터링 (빠름)
  ↓
  임계값 이상만 통과
  ↓
2차: AI 검증 필요성 판단
  - 매칭 키워드 3개 이하
  - 범용 키워드(스포츠, 문화 등) 50% 이상
  ↓
3차: AI 배치 검증 (정확함)
  - 배치로 여러 뉴스 한 번에 검증
  - 신뢰도 70% 이상만 연결
  ↓
최종: 연결
```

**특징:**
- 대부분은 키워드만으로 처리 (빠름)
- 애매한 경우만 AI 검증 (정확함)
- 배치 처리로 토큰 효율성 극대화

### 연결 품질

```sql
-- 이슈별 연결 뉴스 수 확인
SELECT 
    i.title,
    COUNT(n.id) as news_count
FROM issues i
LEFT JOIN news_data n ON n.issue_id = i.id
WHERE i.created_at > NOW() - INTERVAL '1 day'
GROUP BY i.id, i.title
ORDER BY news_count DESC;
```
