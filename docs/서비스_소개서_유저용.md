# 왜난리(WhyNali)

> 요즘 난리, 한눈에.

---

## 왜난리가 필요한가요?

뉴스 앱을 열면 기사가 수백 개입니다.
커뮤니티에는 같은 사건 글이 넘쳐납니다.
그런데 막상 "이 사건이 왜 터진 건지",
"지금 어디까지 왔는지"는 잘 모르겠습니다.

정보는 넘치는데 맥락은 없습니다.
이슈 하나를 제대로 파악하려면
뉴스, 커뮤니티, SNS를 직접 돌아다녀야 합니다.

왜난리는 이 탐색 비용을 줄입니다.

- "지금 상황"과 "왜 이렇게 됐는지"를 한 페이지에서 동시에 확인
- 이슈에 달린 여론 반응과 변화 흐름을 한눈에 파악
- 관련 뉴스 출처 링크를 한 곳에 모아 신뢰성 확보

---

## 왜난리가 하는 일

지금 막 터지는 이슈도 빠르게 반영해 자동으로 업데이트합니다.
흩어진 정보를 모아 이슈의 흐름을 **발단 → 전개 → 파생 → 종결** 순서로 정리합니다.

한 페이지에서 이걸 확인할 수 있습니다.

- 이 사건이 어떻게 시작됐는지
- 어떻게 번졌는지
- 지금 어떤 상태인지
- 사람들이 어떻게 반응하고 있는지

## 주요 기능

### 실시간 인기 이슈 키워드

검색창을 누르면 지금 가장 뜨거운 이슈 키워드 5개를 바로 확인할 수 있습니다.
키워드를 클릭하면 해당 키워드로 검색 결과를 바로 보여줍니다.

### 검색

원하는 이슈를 키워드로 직접 찾을 수 있습니다.
이슈·투표·커뮤니티 토론 주제를 통합 검색합니다.

### 타임라인

AI가 수집된 뉴스를 분석해 이슈의 흐름을 시간순으로 정리합니다.
발단부터 종결까지 어느 시점에 무슨 일이 있었는지 한눈에 확인할 수 있습니다.

### 이슈 상태

| 상태 | 의미 |
|------|------|
| 점화 중 | 반응이 급격히 늘어나는 이슈 |
| 화제 집중 | 반응이 활발한 이슈 |
| 종결 | 관심이 줄어든 이슈 |

### 감정 표현

좋아요 / 싫어요 / 화나요 / 팝콘각 / 응원 / 애도 / 사이다

이슈마다 내 반응을 남길 수 있습니다.
실시간으로 쌓이는 반응으로 여론의 온도를 확인하세요.

### 투표

"여러분은 어떻게 생각하세요?"

AI가 이슈를 분석해 투표 주제를 생성하고,
운영자가 검토·승인한 이슈에 한해 투표가 열립니다.
같은 이슈에서도 상황이 바뀌면 새로운 투표가 추가될 수 있어
시점별로 여론이 어떻게 달라졌는지 확인할 수 있습니다.

### 댓글

의견을 남기고 다른 사람의 생각을 볼 수 있습니다.
베스트 댓글은 상단에 자동으로 올라옵니다.

### 커뮤니티

단순 반응을 넘어 이슈에 대해 깊게 이야기하고 싶다면 커뮤니티를 이용하세요.

AI가 이슈를 분석해 토론 주제를 생성하고,
운영자가 검토·승인한 주제에 한해 커뮤니티에 등록됩니다.
등록된 주제에 철학적 관점으로 의견을 남길 수 있습니다.

## 이용 안내

이슈·타임라인 조회는 로그인 없이 누구나 볼 수 있습니다.
감정 표현·댓글·투표·커뮤니티 토론은 로그인 후 이용 가능합니다.

| | 비로그인 | 로그인 |
|-|----------|--------|
| 이슈·타임라인 보기 | 가능 | 가능 |
| 감정 표현 | - | 가능 |
| 댓글 작성 | - | 가능 |
| 투표 참여 | - | 가능 |
| 커뮤니티 토론 | - | 가능 |

로그인은 구글·네이버·카카오 중 하나로 바로 할 수 있습니다.
별도 회원가입이 필요 없습니다.
