# AI API 비교 분석 및 추천

왜난리 프로젝트의 서비스 런칭을 위한 AI API 선택 가이드입니다.

## 현재 AI 사용 현황

### 주요 AI 활용 기능

왜난리 프로젝트는 다음 기능에 AI를 사용합니다 (2026.03 기준):

**✅ 활성화된 AI 기능**

1. **AI 카테고리 분류** (`ENABLE_AI_CATEGORY=true`)
    - 8개 카테고리 자동 분류
    - 하루 평균 4개 이슈 × 250토큰 = 1,000토큰/일

2. **AI 커뮤니티 매칭** (`ENABLE_AI_COMMUNITY_MATCHING=true`)
    - 상위 10개만 샘플링하여 AI 검사
    - 도돌이표 문제 해결

3. **AI 뉴스 연결 검증** (`ENABLE_AI_NEWS_VALIDATION=true`)
    - 키워드 매칭 후 AI로 실제 관련도 검증
    - 오연결 방지
    - 하루 약 121,000토큰 (전체의 약 60%)

4. **AI 중복 이슈 감지** (`ENABLE_AI_DUPLICATE_CHECK=true`)
    - 같은 사건을 다르게 표현한 이슈 자동 병합
    - Confidence 80% 이상만 중복 처리

5. **AI 중복 그룹 감지** (`ENABLE_AI_DUPLICATE_GROUP_CHECK=true`)
    - 키워드 그루핑 후 쪼개진 그룹 재검증 및 병합

**❌ 비활성화된 AI 기능**

- **AI 그룹 검증** (`ENABLE_AI_GROUP_VALIDATION=false`)
- **AI 그루핑** (`ENABLE_AI_GROUPING=false`)
    - Perplexity 응답 품질 문제로 비활성화
    - 키워드 방식 사용 중

### 예상 토큰 사용량

환경변수 코멘트와 기능 분석 기준:

- AI 뉴스 검증: 약 121,000 토큰/일 (전체의 약 60%)
- AI 카테고리 분류: 약 1,000 토큰/일 (하루 4개 이슈 × 250토큰)
- AI 커뮤니티 매칭: 샘플링 전략으로 최소화
- AI 중복 체크: 변동성 있음

**일일 예상 사용량: 약 150,000~200,000 토큰**
**월간 예상 사용량: 약 6,000,000 토큰 (6M)**

입력:출력 비율 1:1 가정 시 3M input + 3M output

### 현재 설정

```bash
AI_PROVIDER=groq
GROQ_API_KEY=5개 키 순환 사용
```

현재 Groq 무료 플랜의 TPM(분당 토큰) 6,000 제한을 5개 키 순환으로 우회 중입니다.

## API 비교 분석

### 1. Groq API (Llama 3.1 8B)

#### 가격 (2026년 3월 기준)
- Input: $0.05 / 1M 토큰
- Output: $0.08 / 1M 토큰
- **월간 예상 비용: $0.39** (3M × $0.05 + 3M × $0.08)

#### 장점
- 압도적인 가격 경쟁력 (가장 저렴)
- 초고속 추론 속도 (840 tokens/sec)
- JSON 응답 포맷 지원
- 현재 프로젝트에서 이미 안정적으로 사용 중

#### 단점
- 8B 모델의 제한된 추론 능력
- 복잡한 맥락 이해가 필요한 작업에서 오류율 높음
- 미묘한 카테고리 분류나 중복 판단에서 정확도 한계
- 무료 플랜의 Rate Limit으로 다중 키 관리 필요

#### 적합성
간단한 JSON 파싱, 키워드 기반 매칭에는 충분하지만, 왜난리의 핵심 가치인 이슈 품질 관리에는 한계가 있습니다.

### 2. Claude API (Anthropic)

#### 가격 (2026년 3월 기준)

**Claude Haiku 4.5 (빠른 경량 모델)**
- Input: $1.00 / 1M 토큰
- Output: $5.00 / 1M 토큰
- **월간 예상 비용: $18** (3M × $1 + 3M × $5)

**Claude Sonnet 4.6 (고품질 모델)**
- Input: $3.00 / 1M 토큰
- Output: $15.00 / 1M 토큰
- **월간 예상 비용: $54** (3M × $3 + 3M × $15)

#### 장점
- 우수한 추론 능력 및 맥락 이해
- 200K 토큰 컨텍스트 윈도우 (긴 문서 처리)
- 높은 정확도 (카테고리 분류, 중복 감지)
- 안정적인 엔터프라이즈급 서비스 (SLA 보장)
- JSON 모드 및 구조화된 출력 지원

#### 단점
- Groq 대비 높은 비용 (Haiku 기준 46배)
- API 키 발급 및 프로바이더 구현 필요

#### 적합성
왜난리의 이슈 품질, 카테고리 정확도, 중복 감지 신뢰성을 크게 향상시킬 수 있습니다. Haiku는 가성비가 우수하며, Sonnet은 더 높은 품질이 필요한 경우 선택할 수 있습니다.

### 3. Perplexity API

#### 가격 (2026년 3월 기준)

**Sonar (기본 모델)**
- Input: $1.00 / 1M 토큰
- Output: $1.00 / 1M 토큰
- 추가 요청 비용: $5~14 / 1,000 요청 (검색 컨텍스트에 따라)
- **월간 예상 비용: $216+** (토큰 $6 + 30K 요청 × $7/1K = $216)

**Sonar Pro**
- Input: $3.00 / 1M 토큰
- Output: $15.00 / 1M 토큰
- 추가 요청 비용: 더 높음

#### 장점
- 실시간 웹 검색 기능 통합
- 최신 정보 접근 가능
- 출처 자동 인용

#### 단점
- 매우 높은 비용 (요청당 과금 모델)
- 왜난리는 과거 데이터 분석이 주 업무로 실시간 검색 불필요
- 구조화된 텍스트 분석에 특화된 기능 없음

#### 적합성
왜난리 프로젝트의 요구사항과 맞지 않습니다. 이미 수집된 뉴스와 커뮤니티 데이터를 분석하므로 실시간 검색 기능의 가치가 낮습니다.

## 추천 전략

### 1차 추천: Claude Haiku 4.5 전면 전환

#### 이유
1. **품질 우선**: 왜난리의 핵심 가치는 이슈 큐레이션 품질입니다. 카테고리 오분류, 중복 이슈, 잘못된 뉴스 연결은 사용자 신뢰도를 크게 떨어뜨립니다.

2. **비용 효율성**: 월 $18은 프로덕션 서비스에 부담 없는 수준입니다. Groq 대비 46배 비싸지만, 절대 금액이 매우 낮아 품질 향상 대비 합리적입니다.

3. **안정성**: Groq 무료 플랜의 Rate Limit 관리 부담 제거. 5개 키 순환 시스템 불필요.

4. **확장성**: 200K 컨텍스트로 더 긴 뉴스 본문, 더 많은 커뮤니티 글 동시 분석 가능.

5. **신뢰성**: Anthropic의 엔터프라이즈급 SLA 보장.

#### 구현 방법
```typescript
// lib/ai/claude-provider.ts 구현
// AI_PROVIDER=claude
// CLAUDE_API_KEY=your_key
```

현재 `groq-provider.ts`와 동일한 인터페이스를 구현하여 최소한의 코드 변경으로 전환 가능합니다.

## Claude API 충전식(Prepaid Credits) 시스템

### 작동 방식

Claude API는 선불 충전식(Prepaid Credits) 결제 시스템을 사용합니다. 사용 전에 미리 크레딧을 구매하고, API 호출 시 크레딧이 차감됩니다.

### 크레딧 구매 방법

1. [platform.claude.com](https://platform.claude.com) 로그인
2. Settings > Billing 메뉴 이동
3. "Buy Credits" 버튼 클릭
4. 원하는 금액 입력 및 결제
5. 즉시 사용 가능

### 결제 수단

- 신용카드
- 직불카드
- 선불카드

기타 결제 수단(은행 이체, PayPal 등)은 지원하지 않습니다.

### 크레딧 관리

#### 유효기간
- 구매일로부터 1년
- 연장 불가
- 기간 만료 시 자동 소멸

#### 환불 정책
- 모든 크레딧 구매는 환불 불가
- 신중하게 필요한 만큼만 구매 권장

#### 자동충전 (Auto-reload)
- 크레딧 잔액이 설정한 임계값 이하로 떨어지면 자동 충전
- 서비스 중단 방지
- Billing 페이지에서 설정 가능

#### 과금 정책
- 성공한 API 호출만 과금
- 실패한 요청(Rate Limit, 에러 등)은 무료
- 실시간 사용량 모니터링 가능

### 왜난리 프로젝트 적용 가이드

#### 초기 충전 금액 권장

**옵션 1: 소액 테스트**
- $20 충전 (약 1개월분)
- 실제 사용량 측정 후 조정
- 위험 최소화

**옵션 2: 분기 단위**
- $60 충전 (약 3개월분, 월 $18 기준)
- 자동충전 설정으로 관리 부담 감소
- 장기 안정성 확보

#### 자동충전 설정 권장사항

```
잔액: $10 이하일 때
충전: $20 자동 충전
```

이렇게 설정하면 월 $18 예상 사용량 기준으로 서비스 중단 없이 안정적 운영 가능합니다.

#### 비용 모니터링 전략

1. **일일 체크**: Claude Console Billing 페이지에서 사용량 확인
2. **주간 분석**: 예상 비용 대비 실제 사용량 비교
3. **기능별 측정**: 각 AI 기능(카테고리 분류, 중복 감지 등)별 비용 추적
4. **임계값 설정**: 월 $25 초과 시 알림 또는 일부 기능 비활성화

#### 비용 초과 시 대응

만약 예상보다 비용이 높다면:

1. 고비용 기능 일시 비활성화
2. 일부 기능만 Claude, 나머지는 Groq 유지 (하이브리드)
3. AI 호출 빈도 조정 (샘플링 비율 낮추기)
4. 캐싱 전략 도입 (반복 요청 줄이기)

### API 티어 시스템

Claude API는 사용량에 따른 티어 시스템을 운영합니다:

- **Tier 1**: 기본 (초기)
- **Tier 2**: 누적 $40 충전 후 자동 승급
    - Rate Limit 증가
    - 더 높은 처리량 가능

왜난리 프로젝트는 초기 $20-50 충전으로 곧 Tier 2 도달 예상되며, 이후 서비스 확장 시 안정적인 운영이 가능합니다.

## 결론

**왜난리 프로젝트에는 Claude Haiku 4.5가 최적입니다.**

- Groq는 비용은 저렴하지만 이슈 품질 리스크
- Claude Haiku는 품질과 비용의 최적 균형점
- Claude Sonnet은 더 높은 품질이 필요한 경우 고려
- Perplexity는 왜난리 특성상 부적합

월 $18의 비용으로 이슈 큐레이션 품질, 사용자 신뢰도, 서비스 안정성을 크게 향상시킬 수 있습니다. 서비스 런칭 시 사용자 경험을 최우선으로 고려한다면 Claude Haiku로의 전환을 강력히 추천합니다.
