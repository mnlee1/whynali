# 자동승인 버그 수정 완료

날짜: 2026-03-11
이슈: 화력 30점 미만 + 연예/정치 카테고리 이슈가 자동 승인되는 버그

## 문제 원인

`app/api/cron/recalculate-heat/route.ts`의 화력 재계산 Cron에서 대기 이슈를 자동 승인할 때 **카테고리 체크를 하지 않았습니다**.

### 기존 코드 (버그)

```typescript
// 화력만 체크하고 카테고리는 무시
if (issue.approval_status === '대기') {
    if (heatIndex >= AUTO_APPROVE_THRESHOLD) {
        // 자동 승인 처리 (모든 카테고리 포함)
    }
}
```

이로 인해:
- 연예 카테고리 이슈가 화력 30점 도달 → 자동 승인 (본래는 관리자 승인 필요)
- 정치 카테고리 이슈가 화력 30점 도달 → 자동 승인 (본래는 관리자 승인 필요)
- 사회/기술/스포츠 카테고리도 화력만 30점 이상이면 무조건 승인

## 수정 내용

### 1. 자동 승인 조건 함수 추가

```typescript
function shouldAutoApprove(category: IssueCategory, heatIndex: number): boolean {
    return heatIndex >= AUTO_APPROVE_THRESHOLD && AUTO_APPROVE_CATEGORIES.includes(category)
}
```

### 2. Cron 작업 수정

- 이슈 조회 시 `category` 필드 추가
- 자동 승인 시 `shouldAutoApprove()` 함수로 카테고리 + 화력 모두 체크
- 상태 변경 로그에 카테고리와 화력 정보 포함

### 3. 수정된 로직

```typescript
if (issue.approval_status === '대기') {
    const category = issue.category as IssueCategory
    
    // 자동 승인: 화력 30점 이상 AND 카테고리가 사회/기술/스포츠
    if (shouldAutoApprove(category, heatIndex)) {
        // 자동 승인 처리
        result.statusChanged = '대기 → 승인 (화력 ' + heatIndex + '점, ' + category + ')'
    } else if (heatIndex < MIN_HEAT_TO_REGISTER) {
        // 자동 반려: 화력 15점 미만
    }
    // 그 외: 화력 15-29점이지만 연예/정치 카테고리 → 대기 유지
}
```

## 데이터 복구

### 잘못 승인된 이슈 수정

`scripts/fix-wrong-auto-approved.ts` 스크립트로 7개의 잘못된 자동 승인 이슈를 관리자 승인으로 변경했습니다:

1. 정치 이슈, 화력 18점 → 관리자 승인으로 변경
2. 연예 이슈, 화력 18점 → 관리자 승인으로 변경
3. 기술 이슈, 화력 0점 → 관리자 승인으로 변경
4. 기술 이슈, 화력 1점 → 관리자 승인으로 변경
5. 연예 이슈, 화력 16점 → 관리자 승인으로 변경
6. 기술 이슈, 화력 23점 → 관리자 승인으로 변경
7. 스포츠 이슈, 화력 1점 → 관리자 승인으로 변경

## 자동 승인 규칙 (수정 후)

### 조건

**모두 충족해야 자동 승인:**
1. 화력 30점 이상
2. 카테고리가 `사회`, `기술`, `스포츠` 중 하나

### 관리자 승인이 필요한 경우

- 연예 카테고리 (화력 무관)
- 정치 카테고리 (화력 무관)
- 사회/기술/스포츠지만 화력 30점 미만
