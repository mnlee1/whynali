# 카드뉴스 자동화 파이프라인

왜난리의 주간 핫이슈를 Instagram, Threads에 자동으로 업로드하는 파이프라인입니다.

> 참고: X(Twitter) API는 2024년 이후 유료화되어 현재 비활성화 상태입니다. 재활성화가 필요한 경우 Basic 플랜 ($100/월) 이상 구독이 필요합니다.

## 흐름

```
GitHub Actions (매주 수/토 오전 9시)
  → Supabase에서 TOP 3 이슈 조회
  → Groq AI로 슬라이드 텍스트 생성
  → HTML 템플릿에 주입
  → Playwright로 PNG 이미지 8장 생성
  → Instagram 캐러셀 업로드 (UTM: ?utm_source=instagram&utm_medium=cardnews)
  → Threads 캐러셀 업로드 (UTM: ?utm_source=threads&utm_medium=cardnews)
  # → X(Twitter) 연계 트윗 (현재 미사용 - API 유료화로 인해 비활성화)
```

각 플랫폼별로 자동으로 UTM 파라미터가 추가되어 KPI 대시보드에서 유입 경로를 추적할 수 있습니다.

## 로컬 테스트

### 1. 의존성 설치

```bash
npm install
```

### 2. Playwright 브라우저 설치

```bash
npx playwright install chromium
```

### 3. 환경 변수 설정

`.env.local` 파일에 다음 환경 변수 추가:

```env
# Supabase (필수)
NEXT_PUBLIC_SUPABASE_URL=https://your-project.supabase.co
SUPABASE_SERVICE_ROLE_KEY=your-service-role-key

# Groq AI (필수)
GROQ_API_KEY=your-groq-api-key

# Instagram (선택)
IG_USER_ID=your-instagram-business-user-id
IG_ACCESS_TOKEN=your-instagram-long-lived-access-token

# Threads (선택)
THREADS_USER_ID=your-threads-user-id
THREADS_ACCESS_TOKEN=your-threads-long-lived-access-token

# Twitter (현재 미사용 - API 유료화로 비활성화)
# TWITTER_API_KEY=your-twitter-api-key
# TWITTER_API_SECRET=your-twitter-api-secret
# TWITTER_ACCESS_TOKEN=your-twitter-access-token
# TWITTER_ACCESS_SECRET=your-twitter-access-secret
```

### 4. 테스트 실행 (이미지만 생성)

```bash
npx tsx scripts/card-news/pipeline.ts
```

생성된 이미지는 `scripts/card-news/output/` 폴더에 저장됩니다.

### 5. 실제 업로드 테스트

```bash
npx tsx scripts/card-news/pipeline.ts --publish
```

SNS 환경 변수가 설정된 플랫폼에만 자동으로 업로드됩니다.

## SNS API 설정

### Instagram

1. Instagram 계정을 Business/Creator 계정으로 전환
2. Facebook Page 생성 및 연결
3. [Meta for Developers](https://developers.facebook.com/)에서 앱 생성
4. 필요 권한 신청: `instagram_content_publish`, `pages_read_engagement`
5. 장기 액세스 토큰 발급 (60일 유효)

상세 가이드: `88_Meta_앱_설정_가이드.md` 참고

### Threads

1. Instagram과 동일한 Meta 앱 사용 가능
2. Tech Provider Verification 신청 (약 1주일)
3. 필요 권한 신청: `threads_basic`, `threads_content_publish`
4. 장기 액세스 토큰 발급 (60일 유효)

상세 가이드: `88_Meta_앱_설정_가이드.md` 참고

### Twitter (X) - 현재 미사용

2024년 이후 Twitter API가 유료화되어 현재는 사용하지 않습니다.
Basic tier는 월 $100부터 시작하며, 무료 tier로는 트윗 작성이 불가능합니다.

## GitHub Actions 자동 실행

`.github/workflows/card-news.yml` 파일이 이미 생성되어 있습니다.

### GitHub Secrets 등록

GitHub 저장소 → Settings → Secrets and variables → Actions에서 다음 Secrets 추가:

필수:
- `NEXT_PUBLIC_SUPABASE_URL`
- `SUPABASE_SERVICE_ROLE_KEY`
- `GROQ_API_KEY`

SNS 연동 (선택):
- `IG_USER_ID`
- `IG_ACCESS_TOKEN`
- `THREADS_USER_ID`
- `THREADS_ACCESS_TOKEN`

현재 미사용 (API 유료화로 비활성화):
<!--
- `TWITTER_API_KEY`
- `TWITTER_API_SECRET`

- `TWITTER_ACCESS_SECRET`
-->

### 실행 일정

- 매주 수요일 오전 9시 (KST)
- 매주 토요일 오전 9시 (KST)

### 수동 실행

GitHub 저장소 → Actions → 카드뉴스 자동 생성 및 업로드 → Run workflow

## 주의사항

1. **액세스 토큰 갱신**
   - Instagram/Threads: 60일마다 갱신 필요
   - Twitter: 현재 미사용 (API 유료화로 비활성화)

2. **Rate Limits**
   - Instagram: 문서화되지 않음 (일반적으로 충분)
   - Threads: 250개 게시물/24시간
   - Twitter: 현재 미사용 (API 유료화로 비활성화)

3. **Threads 발행 대기**
   - 캐러셀 컨테이너 생성 후 최소 30초 대기 필수
   - 파이프라인이 자동으로 처리

4. **이미지 URL**
   - Instagram/Threads API는 공개 URL 필요
   - 자동으로 Supabase Storage에 업로드 후 public URL 사용

## 트러블슈팅

### 이미지가 생성되지 않음

```bash
# Playwright 브라우저 재설치
npx playwright install chromium --force
```

### Instagram/Threads 업로드 실패

- 액세스 토큰이 만료되었는지 확인
- 이미지가 Supabase Storage에 정상 업로드되었는지 확인
- Supabase `public` 버킷이 public 읽기 권한으로 설정되었는지 확인

### Twitter 트윗 실패 - 현재 미사용

현재 Twitter API 유료화로 인해 X(Twitter) 연동 기능은 비활성화되어 있습니다.
재활성화가 필요한 경우 유료 플랜 구독이 필요합니다 (Basic $100/월 이상).
