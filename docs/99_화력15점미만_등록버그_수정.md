## 문제 발견

### 심각한 버그
**등록 시점부터 화력 6-7점인 이슈 17건 발견**
- 등록 기준: 화력 15점 이상
- 실제: 6-7점으로 기준 미달인데 등록됨

### 영향 범위
```
전체 이슈 53건 중:
- 화력 15점 미만: 23건 (43%)
  - 등록 후 하락: 6건 (정상)
  - 등록 시부터 미달: 17건 (버그)
```

**수정 후:**
```typescript
const heatIndex = await calculateHeatIndex(newIssue.id).catch(() => 0)

// 화력 15점 미만이면 이슈 삭제
const MIN_HEAT_TO_REGISTER = parseInt(process.env.CANDIDATE_MIN_HEAT_TO_REGISTER || '15')
if (heatIndex < MIN_HEAT_TO_REGISTER) {
    console.log(`❌ [화력 부족] "${issueTitle}" (화력: ${heatIndex}점) - 이슈 삭제`)
    await supabaseAdmin
        .from('issues')
        .delete()
        .eq('id', newIssue.id)
    continue
}

await supabaseAdmin
    .from('issues')
    .update({ 
        heat_index: heatIndex,
        created_heat_index: heatIndex
    })
    .eq('id', newIssue.id)

console.log(`✅ [긴급 이슈 생성] "${issueTitle}" (화력: ${heatIndex}점)`)
```
