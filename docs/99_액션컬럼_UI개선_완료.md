# 이슈 관리 페이지 액션 컬럼 UI 개선

> 개선일: 2026-03-11
> 파일: `app/admin/issues/page.tsx`

## 개선 목적

이슈 목록에서 이슈 정보를 한눈에 파악하기 어려운 문제 해결
- 액션 컬럼이 너무 넓어서 (256px) 공간 낭비
- 텍스트 버튼이 가독성을 방해

## 개선 내역

### 1. 컬럼 너비 축소 ✅

**변경 전:**
```typescript
<th className="px-4 py-3 ... w-64">  // 256px
    액션
</th>
```

**변경 후:**
```typescript
<th className="px-2 py-3 ... w-24">  // 96px
    액션
</th>
```

**절약된 공간:** 160px (62% 감소)

### 2. 텍스트 버튼 → 아이콘 버튼 ✅

**변경 전:**
```
[미리보기] [승인] [반려]  // 3개 버튼 = ~200px
```

**변경 후:**
```
[👁️] [✓] [✕]  // 3개 아이콘 = ~60px
```

## 아이콘 버튼 구성

### 공통 버튼

| 아이콘 | 기능 | 색상 | 툴팁 |
|--------|------|------|------|
| 👁️ (eye) | 미리보기 | 파란색 | "미리보기" |

### 상태별 버튼

**대기 상태:**
- ✓ (check) - 승인 (초록색)
- ✕ (x) - 반려 (빨간색)

**승인 상태:**
- ✕ (x) - 반려 (빨간색)

**반려 상태:**
- ↻ (rotate) - 복구 (회색)

**병합됨 상태:**
- ↗ (arrow) - 병합된 이슈 보기 (보라색)

### 버튼 스타일

```typescript
className="p-1.5 text-{color}-600 hover:bg-{color}-50 rounded transition-colors"
```

**특징:**
- 16px 아이콘 (`w-4 h-4`)
- 6px 패딩 (`p-1.5`)
- 호버 시 배경색 변경
- 부드러운 전환 효과

## 접근성 개선

### 툴팁 추가

모든 아이콘 버튼에 `title` 속성으로 설명 제공:

```typescript
<button title="미리보기">...</button>
<button title="승인">...</button>
<button title="반려">...</button>
<button title="복구">...</button>
<a title="병합된 이슈 보기">...</a>
```

### 시각적 피드백

```typescript
// 호버 효과
hover:bg-{color}-50

// 전환 효과
transition-colors
```

## 효과 측정

### 공간 절약

| 항목 | 변경 전 | 변경 후 | 절약 |
|------|---------|---------|------|
| 액션 컬럼 너비 | 256px (w-64) | 96px (w-24) | 160px |
| 버튼당 공간 | ~80px | ~20px | ~60px |
| 총 절약 | - | - | **62%** |

### 가독성 향상

**변경 전:**
```
제목 (짧게 보임)          [미리보기] [승인] [반려]
```

**변경 후:**
```
제목이 더 길게 보여서 한눈에 파악 가능    [👁️][✓][✕]
```

### 사용자 경험

1. **이슈 제목이 더 길게 표시됨**
   - 160px 추가 공간 확보
   - 제목을 끝까지 읽을 수 있음

2. **뱃지와 아이콘이 명확하게 보임**
   - 🔥 급증 아이콘
   - ⚠️ 즉시 처리 필요 아이콘
   - "즉시 처리 필요" 뱃지

3. **스크롤 감소**
   - 한 화면에 더 많은 정보 표시
   - 가로 스크롤 필요성 감소
