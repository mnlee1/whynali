# 왜난리 프로젝트 기능 테스트 보고서

**테스트 일시**: 2026-03-05  
**테스트 환경**: localhost:3000 (개발 서버)  
**테스트 방법**: API 기반 테스트 + E2E 스크립트 생성

## 1. 테스트 개요

왜난리 프로젝트의 주요 기능들을 체계적으로 테스트하여 현재 구현 상태와 작동 여부를 확인했습니다.

## 3. 발견된 문제점

### 3.1 API 라우트 이슈

다음 API 엔드포인트가 404를 반환합니다:

1. `/api/categories/stats` - 카테고리별 통계 (구현 필요)

### 3.2 검색 기능 이슈

- `/search?q=테스트` 페이지가 400 오류 반환
- 원인: URL 인코딩 문제 (한글 검색어 깨짐)
- 검색 API 자체는 정상 작동 (`/api/search?q=스포츠` 테스트 성공)
- 해결 필요: 검색 페이지에서 한글 URL 인코딩 처리

### 3.3 데이터 상태

**있는 데이터**:
- 이슈 목록 (13개)
- 타임라인 데이터 (이슈별 5개 항목)
- 토론 주제 (6개)

**없는 데이터**:
- 테스트한 특정 이슈에 투표 데이터 없음
- 댓글 데이터 없음
- 감정 표현 데이터 없음

## 4. 정상 작동 확인 기능

- 이슈 목록 조회 정상 (13개 이슈)

- 타임라인 데이터 구조 정상:
    - 발단, 전개, 파생 단계별 구분

    - 시간순 정렬

- 토론 주제 API 정상 작동 (`/api/discussions`)
- 토론 주제 6개 확인
- AI 생성 주제 및 사용자 작성 주제 혼합

### 4.4 카테고리 페이지

- 연예, 스포츠, 정치, 사회, 기술 카테고리 페이지 모두 접근 가능
- 카테고리별 필터링 API 정상 작동

### 4.5 기본 레이아웃

- 헤더, 푸터 정상 렌더링

### 4.6 투표, 댓글, 감정 표현 API

## 5. E2E 테스트 스크립트

### 5.1 생성된 테스트 파일

- `e2e/02-category.spec.ts` - 카테고리 필터링 (5개 테스트)
- `e2e/03-issue-detail.spec.ts` - 이슈 상세 페이지 (5개 테스트)
- `e2e/04-vote.spec.ts` - 투표 기능 (3개 테스트)
- `e2e/05-comments.spec.ts` - 댓글 기능 (4개 테스트)
- `e2e/06-community.spec.ts` - 커뮤니티 (3개 테스트)
- `e2e/07-search.spec.ts` - 검색 기능 (3개 테스트)
- `e2e/08-navigation.spec.ts` - 네비게이션 (3개 테스트)

**총 30개 테스트 케이스**

### 5.2 E2E 테스트 실행 이슈

Playwright 실행 시 브라우저 바이너리 경로 문제로 테스트 실패:
```
Executable doesn't exist at /var/folders/.../playwright/chromium_headless_shell-1208/chrome-headless-shell-mac-arm64/chrome-headless-shell
```

**해결 방법**:
로컬 터미널에서 직접 실행:
```bash
cd whynali
npx playwright install chromium
npx playwright test
```
