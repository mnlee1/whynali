# 이슈 후보 알림 기준 미달 버그 분석

## 문제 상황

**캡처 이미지:**
- 제목: "고려대학, '인터배터리 2026'서 이차전지 소재 전방물질 기술력 선보인다"
- 표시: "최근 3시간 뉴스 3건 (4건 → 1건 + 커뮤니티 0)"

**문제점:**
- `ALERT_THRESHOLD = 5` (최소 5건 필요)
- 실제 건수: 3-4건 (커뮤니티 0건)
- **기준 미달인데 알림창에 노출됨**

## 원인 분석

### 1. `recentCount` vs 실제 표시 건수 불일치

코드를 확인한 결과, `recentCount`와 실제 알림에 표시되는 건수가 다를 수 있는 시나리오 발견:

```typescript
// 739번 라인: ALERT_THRESHOLD 체크
if (recentCount < ALERT_THRESHOLD) continue

// 846번 라인: alerts에 추가
result.alerts.push({
    title: representativeTitle,
    count: recentCount,      // ← 그루핑 당시 건수
    newsCount: newsIds.length,  // ← 실제 뉴스 건수
    communityCount: communityIds.length,
})
```

**`recentCount`는 그루핑 당시의 전체 아이템 수**이지만, **실제 뉴스 건수(`newsIds.length`)는 다를 수 있습니다.**

### 2. 가능한 원인 시나리오

#### 시나리오 A: 중복 그루핑 후 일부 제거
```typescript
// 598-701번 라인: AI 중복 그룹 체크 및 병합
if (enableDuplicateCheck && groups.length >= 2) {
    // 그룹 병합 로직
    groups = groups.filter(g => g.items.length > 0)
}
```

**가능성:**
1. 초기 그루핑: Group A (3건) + Group B (2건) = 5건
2. 739번 라인 체크: `recentCount = 5` → 통과
3. AI 중복 체크: Group B의 2건이 Group A에서 제거됨 (중복으로 판정)
4. 최종: `newsIds.length = 3`이지만 `recentCount = 5`로 유지

하지만 이것은 **병합**이지 제거가 아니므로 건수가 줄어들지 않습니다.

#### 시나리오 B: 뉴스 건수와 커뮤니티 건수의 혼동
```typescript
// 737번 라인
const recentCount = group.items.length

// group.items에는 뉴스와 커뮤니티가 모두 포함될 수 있음
interface RawItem {
    type: 'news' | 'community'
}
```

**문제점 발견!**
- `recentCount`는 **뉴스 + 커뮤니티 전체 건수**
- 하지만 뉴스 그루핑 시에는 **뉴스만 그루핑**됨
- 커뮤니티는 나중에 키워드 매칭으로 추가됨

```typescript
// 552-566번 라인: 데이터 조회
const [{ data: newsItems }, { data: communityItems }] = await Promise.all([
    supabaseAdmin
        .from('news_data')
        .select('id, title, created_at, category, source')
        .is('issue_id', null)
        .gte('created_at', sinceWindow),  // 3시간 창
    supabaseAdmin
        .from('community_data')
        .select('id, title, created_at')
        .is('issue_id', null)
        .gte('created_at', since24h),     // 24시간 창
])

// 568-574번 라인: 뉴스만 그루핑
const newsRawItems: RawItem[] = (newsItems ?? []).map((n) => ({
    ...n, type: 'news' as const
}))

if (newsRawItems.length === 0) {
    return { created: 0, alerts: [], evaluated: 0 }
}

// 592-595번 라인: 뉴스만으로 그루핑
groups = groupItems(newsRawItems)
```

**결론:** `group.items`에는 **뉴스만** 포함되므로 `recentCount`는 뉴스 건수입니다.
