## 2. 로직 요약

- **등록:** 화력 15점 미만이면 이슈 삭제(트랙 A·급증 동일). 등록 시 `created_heat_index` 한 번 저장, 이후 변경 없음.
- **등록 후:** 화력 하락해도 자동 반려 없음. 대기 유지. 자동 승인은 30점 이상+사회/기술/스포츠만.
- **레거시:** `approval_status=반려` + `approval_type=auto` = 예전 자동 반려 로직으로 저장된 데이터. `scripts/delete_legacy_auto_rejected_issues.ts`로 삭제 가능.

---

## 3. 레거시 삭제 실행

```bash
npx tsx scripts/delete_legacy_auto_rejected_issues.ts
```

대상 목록 출력 후 터미널에 `yes` 입력 시 삭제.
