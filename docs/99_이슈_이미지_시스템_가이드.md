# 왜난리 이슈 이미지 시스템 가이드

## 개요

이슈 대표 이미지는 **Pexels** 스톡 사진으로 자동·수동 검색합니다.
관리자는 최대 3장 중 대표를 고르거나 **이미지 재검색**으로 교체합니다.

AI 생성 썸네일(Vertex / HuggingFace) 기능은 **2026-05 제거**되었습니다.

---

## 데이터

| 컬럼 | 설명 |
|------|------|
| `thumbnail_urls` | 이미지 URL 배열 (최대 3개) |
| `primary_thumbnail_index` | 대표 인덱스 (0~2, -1이면 그라데이션) |

타입: `types/issue.ts`

---

## 이미지가 붙는 시점

### 1. 트랙 A (자동)

`app/api/cron/track-a/route.ts` → `fetchPexelsImages(title, category)`

이슈 등록 직후 썸네일 3장 저장.

### 2. 관리자 승인 (백업)

`app/api/admin/issues/[id]/approve/route.ts`

`thumbnail_urls`가 **비어 있을 때만** Pexels 검색.

### 3. 관리자 미리보기 (수동 재검색)

`components/admin/IssuePreviewDrawer.tsx` → **이미지 재검색**

`POST /api/admin/issues/[id]/refresh-thumbnails`

- Pexels에서 새 3장 검색
- 기존 URL **전체 교체**
- `primary_thumbnail_index = 0`
- 재검색 시 `seed`로 후보 다양화

---

## 관리자 UI

**이슈 관리 → 미리보기 → 대표 이미지**

| 버튼 | 동작 |
|------|------|
| 이미지 재검색 | Pexels 3장 새로 받기 |
| 그라데이션 배경 사용 | 대표 해제 (`primary_thumbnail_index = -1`) |
| 라디오 선택 | 대표 이미지 변경 (`primary-thumbnail` API) |

---

## API

| 경로 | 메서드 | 설명 |
|------|--------|------|
| `/api/admin/issues/[id]/refresh-thumbnails` | POST | Pexels 재검색 |
| `/api/admin/issues/[id]/primary-thumbnail` | POST | 대표 인덱스 변경 |
| `/api/admin/issues/[id]/remove-thumbnails` | POST | 대표 해제 (URL 유지) |

## 구현 파일

|------|------|
| `lib/pexels.ts` | Groq 키워드 추출 + Pexels 검색 (만료 Groq 키 건너뜀) |

---

## 환경 변수

| 변수 | 필수 | 설명 |
|------|------|------|
| `PEXELS_API_KEY` | **예** | Pexels API |
| `GROQ_API_KEY` | 권장 | 검색 키워드 추출 (없으면 카테고리 폴백) |

**더 이상 사용하지 않음** (Vercel에서 삭제 가능):

- `VERTEX_PROJECT_ID`, `VERTEX_LOCATION`
- `HF_TOKEN`

---

## 레거시 URL

DB에 `images.unsplash.com` 등 예전 URL이 남아 있어도 표시는 됩니다.
일괄 교체: `scripts/fix-all-production-images.ts`, `scripts/fix-dev-images.ts`

---

## 장애 대응

| 증상 | 원인 | 조치 |
|------|------|------|
| 재검색 실패 alert | `PEXELS_API_KEY` 없음/오류 | 키 확인 |
| 항상 비슷한 3장 | 동일 검색어 | 재검색 재시도 (seed 변경됨) |
| Groq 키워드 실패 | Groq 키 만료 | 키 갱신 또는 카테고리 폴백만 사용 |
