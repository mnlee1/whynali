### 결정 배경

- **펨코·엠팍**: robots.txt 우회는 정보통신망법 위반 가능성(형사) — 절대 제외
- **인스티즈**: 더쿠와 이용층·주제 90% 중복 → 화력 지수 왜곡만 발생, 가치 없음 → 제외
- **클리앙·보배드림·루리웹·뽐뿌**: 약관상 리스크는 민사 수준이며, 탐지 가능성이 낮고 메타데이터만 수집하므로 사업적 판단으로 리스크 감수 결정. 각 커뮤니티가 더쿠·네이트판과 겹치지 않는 독립적 인구층을 커버하여 화력 감지 품질 향상에 기여.

## 2. 사이트별 상세 검토

### 2.1 펨코 (fmkorea.com)

**판정: ❌ 수집 불가**

| 항목 | 내용 |
|------|------|
| robots.txt | `User-agent: * / Disallow: /*` — 전체 경로 차단 |
| 약관 크롤링 금지 | 별도 명시 없음 (robots.txt로 이미 기술적 차단) |
| 리스크 수준 | **높음** |

**판단 근거:**
- robots.txt에 `Disallow: /*` 명시 = 모든 봇·크롤러에 전체 경로 수집 금지
- robots.txt 무시 시 한국 법원 기준으로 부정경쟁행위·정보통신망법 위반 가능성
- 대법원 2022도1533: robots.txt 차단을 우회한 크롤링은 정보통신망법 위반 가능

### 2.3 클리앙 (clien.net)

**판정: ❌ 수집 불가**

| 항목 | 내용 |
|------|------|
| robots.txt | 부분 허용 (Disallow 없음) |
| 약관 금지 조항 | **제13조: "무단 수집·복제·배포 금지" 명시** |
| 리스크 수준 | **높음** |

**약관 핵심 조항:**
```
제13조 (저작권 등)
회원이 서비스 내에 게시한 게시물의 지식재산권 및 초상권 등은
해당 저작자에게 귀속됩니다. 단, 회사는 서비스의 운영, 전시,
전송, 배포, 홍보의 목적으로 별도 허락 없이 무상으로 사용할 수 있습니다.
회원은 다음 각 호의 행위를 하여서는 안됩니다.
 → 서비스 내 정보를 회사의 사전 승낙 없이 수집, 복제, 배포, 제3자 제공
```

**판단 근거:**
- "무단 수집" 행위 자체를 약관에서 명시적으로 금지
- 메타데이터도 "서비스 내 정보"에 해당할 여지 있음
- 약관 위반 = 민사 손해배상 청구 가능

### 2.6 루리웹 (ruliweb.com)

**판정: ⚠️ 그레이존 — 수집 보류 권고**

| 항목 | 내용 |
|------|------|
| robots.txt | 직접 확인 불가 (서버 타임아웃) — 브라우저 직접 접속 필요 |
| 약관 금지 조항 | 복제·제3자 제공 금지 조항 있음 (크롤링 직접 명시 여부 불명확) |
| 리스크 수준 | **중간** |

**약관 확인 내용 (간접 확인, 2022.09.17 개정 기준):**
```
회원은 다음 각 호의 행위를 하여서는 안됩니다.
 → 서비스를 이용하여 얻은 정보를 회사의 사전 승낙 없이 영리 또는 비영리의
    목적으로 복제, 출판, 방송 등에 사용하거나 제3자에게 제공하는 행위
```

**확인 필요 사항:**
- `https://www.ruliweb.com/robots.txt` 브라우저 직접 접속 확인
  - `Disallow: /*` 이면 ❌ 수집 불가
  - 허용 경로 있으면 ⚠️ 약관 조항만 쟁점

**법적 판단:**
- 약관에 "복제, 출판, 방송 사용 또는 제3자 제공" 금지 조항이 있음
- "서비스를 이용하여 얻은 정보"의 범위에 메타데이터가 포함될 여지 존재
- 단, 크롤링·자동수집을 명시적으로 금지하지 않아 법원에서 다툴 여지는 있음
- 안전 설계 원칙 준수 측면에서 **수집 보류가 적절**

**판정: ⚠️ 그레이존 — 수집 보류 권고**

| 항목 | 내용 |
|------|------|
| robots.txt | `/zboard` 경로 허용 (간접 확인) — 기술적 차단 없음 |

| 리스크 수준 | **중간** |

**robots.txt 간접 확인 내용:**
```
User-agent: *
Allow: /zboard
(게시판 경로 Disallow 없음 — 기술적 차단 없음)
```

**약관 확인 내용 (간접 확인):**
```
회원은 서비스로 얻은 정보를 영리·비영리 목적으로
무단 출판·복제하거나 제3자에게 제공해서는 안 됩니다.
(개인정보 무단 수집·유포 금지 조항 별도 존재)
```

**법적 판단:**
- robots.txt 기준으로는 기술적 차단 없음 (수집 가능)
- 그러나 약관에 복제·제3자 제공 금지 조항이 있어 약관상 쟁점 존재
- 메타데이터만 수집 시 실질적 저작권 침해 가능성은 낮으나
- 현재 서비스의 보수적 법적 설계 원칙상 **수집 보류가 적절**

## 5. 구현 시 주의사항

### 5.1 커뮤니티 추가 시 반드시 확인할 체크리스트

```
□ robots.txt 직접 확인 (브라우저에서 /robots.txt 접속)
□ 이용약관 전문 확인 (크롤링, 자동수집, 무단 수집 관련 조항)
□ 개인정보처리방침 확인
□ 명시적 금지 조항 없음 확인
□ 수집 주기 완화 (10분 이상 간격)
□ User-Agent에 서비스명 및 연락처 명시
□ 메타데이터만 수집 (본문, 닉네임, 이미지 제외)
□ 원문 링크 제공 확인
```
