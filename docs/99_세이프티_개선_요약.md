## 무료로 구현된 이유

모든 개선 사항은 **기존 무료 인프라만 활용**:

1. **단어 경계 매칭**: 정규식 처리만 추가 (Node.js 내장)
2. **관리자 알림**: 기존 이메일 함수 재사용 + 메모리 쿨다운
3. **Rate Limit 정리**: Vercel Cron 무료 플랜 (월 100회 = 4,320회)
4. **DB 검증 통합**: 기존 Supabase 쿼리 활용

**추가 비용 발생 없음** ✅

## 테스트 시나리오

### 1. 단어 경계 매칭
```
금칙어: "나쁨"

✅ "이건 나쁨" → 차단 (공백 경계)
✅ "이건나쁨" → 차단 (문장 경계)
✅ "남바다" → 통과 (다른 단어 일부)
```

### 2. 관리자 알림
```
1. 금칙어 포함 댓글 10개 작성
2. 관리자 이메일 수신 확인
3. 10분 내 재작성 시 중복 알림 없음 확인
```

### 3. 토론 댓글 수정
```
1. 토론 댓글 작성 (정상)
2. 금칙어 포함으로 수정 시도
3. "검토 대기" 상태로 전환 확인
```
