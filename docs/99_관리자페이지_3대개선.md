# 관리자 페이지 3대 개선사항

날짜: 2026-03-13

## 문제 발견

사용자가 발견한 3가지 문제:
1. 관리자 반려 처리 안 했는데 "관리자 반려"로 표시
2. 화력 15점 미만인데 이슈 등록됨
3. 카테고리 오분류 (WBC 이슈 → IT과학)

## 1. 반려 표시 오류 수정

### 문제
- 실제 관리자 반려 기록: 0개
- 모든 반려 이슈가 "관리자 반려"로 표시
- 실제로는 `approval_type: 'auto'` (자동 반려)

### 원인
**파일**: `app/admin/issues/page.tsx` (180-184번 라인)

```typescript
if (issue.approval_status === '반려') {
    return {
        label: '관리자 반려',  // ❌ 무조건 관리자 반려
        className: 'bg-red-100 text-red-700 border-red-200'
    }
}
```

### 해결
```typescript
if (issue.approval_status === '반려') {
    // approval_type으로 자동/관리자 구분
    if (issue.approval_type === 'auto') {
        return {
            label: '자동 반려',
            className: 'bg-gray-100 text-gray-700 border-gray-200'
        }
    } else {
        return {
            label: '관리자 반려',
            className: 'bg-red-100 text-red-700 border-red-200'
        }
    }
}
```

## 2. 화력 미달 이슈 등록 문제

### 문제
5개 이슈가 화력 15점 미만인데 승인 상태:
- 블랙핑크 지수: 0점
- 이탈리아 WBC: 0점
- 피는 못 속여: 0점
- WBC 한국 8강: 1점
- 올림픽 메달: 1점

모두 `source_track: 'manual'` (수동 생성)

### 원인
**파일**: `app/api/issues/route.ts` (108번 라인)

```typescript
// 화력 계산 (수동 생성 이슈는 화력 체크 안 함 - 관리자가 직접 생성)
// 화력 계산 로직이 있다면 여기서 실행
```

수동 생성 이슈는 화력을 계산하지 않음 → 0점으로 생성됨

### 해결
```typescript
// 화력 계산 (수동 생성 이슈도 계산)
const { calculateHeatIndex } = await import('@/lib/analysis/heat')
try {
    await calculateHeatIndex(newIssue.id)
} catch (heatError) {
    console.error('화력 계산 실패:', heatError)
    // 화력 계산 실패해도 이슈 생성은 성공
}
```

**효과**: 이제 수동 생성 이슈도 연결된 뉴스/커뮤니티 기반으로 화력 계산

## 3. 카테고리 AI 분류 개선

### 문제
"WBC 점수 조작 죄송" 이슈가 **IT과학**으로 분류됨
- 올바른 카테고리: 경제 (기업 마케팅 논란) 또는 사회

### 원인
**파일**: `app/api/cron/track-a/route.ts` (581-589번 라인)

AI 프롬프트의 카테고리 설명이 불충분:
- "IT과학" 카테고리 누락
- 카테고리 구분 기준 모호
- 우선순위 없음

### 해결

#### A. AI 프롬프트 개선

```typescript
## 3단계: 카테고리 분류 (정확히 판단)
다음 중 하나 선택:
- "사회": 사건사고, 법원 판결, 복지, 교육, 사회 이슈
- "정치": 정치인, 정부 정책, 국회, 선거, 외교
- "연예": 연예인, 아이돌, 드라마, 영화, 음악, 방송
- "스포츠": 스포츠 경기, 선수, 대회, 올림픽, 월드컵
- "경제": 기업, 주식, 부동산, 금융, 경제 정책
- "IT과학": IT, 과학, 기술, 게임, AI, 스타트업
- "생활문화": 여행, 음식, 패션, 트렌드, 라이프스타일
- "세계": 국제 뉴스, 외국 사건, 해외 이슈

⚠️ 카테고리 판단 우선순위:
1. 스포츠 관련 (경기, 선수) → "스포츠" (예: WBC는 스포츠)
2. 기업 비즈니스 관련 → "경제" (예: 두끼 마케팅은 경제)
3. 연예인/드라마/영화 → "연예"
4. 사건사고/법원/교육 → "사회"
5. 정치인/정책 → "정치"
6. IT/과학/기술 → "IT과학"
7. 불명확하면 → "사회"
```

#### B. 유효성 검사 수정

```typescript
const validCategories: IssueCategory[] = [
    '사회', '정치', '연예', '스포츠', '경제', 'IT과학', '생활문화', '세계'
]
```

#### C. 기존 이슈 수정

WBC 이슈: IT과학 → 경제

3. 오분류 패턴 분석 후 프롬프트 개선
