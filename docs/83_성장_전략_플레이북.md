### 2.1 공유 기능 강화 (최우선)

**문제:** 현재 공유 버튼이 없음

**해결:**

1. 이슈 페이지에 공유 버튼 추가
2. 공유 시 예쁜 미리보기 이미지(OG Image) 자동 생성
3. 공유 추적 (어디로 얼마나 공유되는지)

**구현 위치:**
- `app/issue/[id]/page.tsx` - OG Meta 태그
- `components/issue/ShareButton.tsx` - 공유 버튼 컴포넌트

**공유 버튼 예시:**

```typescript
// components/issue/ShareButton.tsx
'use client'

import { useState } from 'react'

export default function ShareButton({ issueId, title }: { issueId: string, title: string }) {
    const [copied, setCopied] = useState(false)
    
    const shareUrl = `https://whynali.com/issue/${issueId}`
    
    const handleShare = async (platform: string) => {
        // 공유 이벤트 트래킹
        if (typeof window !== 'undefined' && window.gtag) {
            window.gtag('event', 'share', {
                method: platform,
                content_type: 'issue',
                item_id: issueId,
            })
        }
        
        const urls: Record<string, string> = {
            twitter: `https://twitter.com/intent/tweet?text=${encodeURIComponent(title)}&url=${encodeURIComponent(shareUrl)}`,
            facebook: `https://www.facebook.com/sharer/sharer.php?u=${encodeURIComponent(shareUrl)}`,
            kakaotalk: `https://story.kakao.com/share?url=${encodeURIComponent(shareUrl)}`,
        }
        
        if (urls[platform]) {
            window.open(urls[platform], '_blank', 'width=600,height=400')
        }
    }
    
    const copyLink = async () => {
        await navigator.clipboard.writeText(shareUrl)
        setCopied(true)
        setTimeout(() => setCopied(false), 2000)
    }
    
    return (
        <div className="flex gap-2">
            <button
                onClick={() => handleShare('twitter')}
                className="px-4 py-2 bg-blue-400 text-white rounded hover:bg-blue-500"
            >
                트위터 공유
            </button>
            <button
                onClick={() => handleShare('facebook')}
                className="px-4 py-2 bg-blue-600 text-white rounded hover:bg-blue-700"
            >
                페이스북 공유
            </button>
            <button
                onClick={() => handleShare('kakaotalk')}
                className="px-4 py-2 bg-yellow-400 text-gray-900 rounded hover:bg-yellow-500"
            >
                카카오톡 공유
            </button>
            <button
                onClick={copyLink}
                className="px-4 py-2 bg-gray-200 text-gray-900 rounded hover:bg-gray-300"
            >
                {copied ? '복사됨!' : '링크 복사'}
            </button>
        </div>
    )
}
```

### 2.2 알림 시스템 (재방문 유도)

**목표: 재방문율 50% 이상**

**알림 트리거:**
1. 내가 댓글 단 이슈에 새 댓글 달림
2. 내가 반응한 이슈가 종결됨
3. 새 핫이슈 등록 (화력 70+)
4. 내 댓글에 좋아요 10개 이상

**구현 방법:**

**방법 1: 이메일 (Resend 무료 티어)**

```typescript
// lib/notifications/email.ts
import { Resend } from 'resend'

const resend = new Resend(process.env.RESEND_API_KEY)

export async function sendIssueUpdateEmail(
    to: string,
    issueTitle: string,
    issueUrl: string
) {
    await resend.emails.send({
        from: 'no-reply@whynali.com',
        to,
        subject: `[왜난리] "${issueTitle}" 이슈에 새 댓글이 달렸습니다`,
        html: `
            <h2>새 댓글 알림</h2>
            <p>관심있는 이슈에 새로운 댓글이 달렸습니다.</p>
            <p><strong>${issueTitle}</strong></p>
            <a href="${issueUrl}">이슈 보러가기</a>
        `,
    })
}
```

**방법 2: 브라우저 푸시 알림 (무료)**

```typescript
// lib/notifications/push.ts
export async function requestNotificationPermission() {
    if ('Notification' in window && Notification.permission === 'default') {
        const permission = await Notification.requestPermission()
        return permission === 'granted'
    }
    return false
}

export function showNotification(title: string, body: string, url: string) {
    if ('Notification' in window && Notification.permission === 'granted') {
        const notification = new Notification(title, {
            body,
            icon: '/icon-192.png',
            badge: '/badge-72.png',
        })
        
        notification.onclick = () => {
            window.open(url, '_blank')
        }
    }
}
```

### 2.4 SNS 마케팅 전략

**트위터 전략:**

1. 매일 화력 70+ 이슈 자동 트윗
2. 베스트 댓글 하이라이트 (주 1회)
3. 핫한 논란 요약 스레드 (주 1회)
4. 관련 트렌드 검색해서 댓글 달기

**자동 트윗 예시:**

```typescript
// scripts/auto-tweet.ts
import { TwitterApi } from 'twitter-api-v2'

const client = new TwitterApi({
    appKey: process.env.TWITTER_API_KEY!,
    appSecret: process.env.TWITTER_API_SECRET!,
    accessToken: process.env.TWITTER_ACCESS_TOKEN!,
    accessSecret: process.env.TWITTER_ACCESS_SECRET!,
})

export async function tweetHotIssue(issue: Issue) {
    const text = `
🔥 지금 가장 뜨거운 이슈!

"${issue.title}"

화력: ${issue.heat}/100
반응: ${issue.reaction_count}개
댓글: ${issue.comment_count}개

자세히 보기 👉 https://whynali.com/issue/${issue.id}

#왜난리 #이슈 #논란
    `.trim()
    
    await client.v2.tweet(text)
}
```

**인스타그램 전략:**

1. 화력 높은 이슈를 카드뉴스로 제작
2. 스토리에 투표 기능 활용
3. 릴스로 이슈 요약 영상 (1분)

### 2.5 커뮤니티 활성화

**목표: 자발적인 콘텐츠 생산**

**전략:**

1. 베스트 댓글 이벤트 (주간)
   - 가장 많은 좋아요 받은 댓글 선정
   - 상품권 1만원 (또는 배지)

2. 이슈 예측 이벤트 (월간)
   - "다음 주 가장 뜨거울 이슈는?"
   - 맞춘 사람에게 리워드

3. 토론 주제 제안 이벤트
   - 유저가 토론 주제 제안
   - 채택되면 배지 부여
