## 부록: 계산 도구

### A. 복리 성장 계산기

```
현재 가입자: A
주간 성장률: r% (소수점 형태, 예: 10% = 0.10)
기간: n주

최종 가입자 = A × (1 + r)^n

예시:
50명, 주 20% 성장, 8주
= 50 × (1.20)^8
= 50 × 4.30
= 215명
```

### B. 주차별 목표 생성기

```python
# Python으로 쉽게 계산
current = 22  # 현재 가입자
growth = 0.35  # 주간 성장률 (35%)
weeks = 8

for week in range(1, weeks + 1):
    target = int(current * (1 + growth) ** week)
    print(f"{week}주차: {target}명")
```

### C. 성장률 역계산

```
목표를 먼저 정했을 때 필요한 성장률 계산:

필요 성장률 = (목표 / 현재)^(1/주차) - 1

예시:
현재 22명 → 목표 243명 (8주)
= (243 / 22)^(1/8) - 1
= (11.05)^(0.125) - 1
= 1.35 - 1
= 0.35 = 35% 주간 성장률 필요
```
