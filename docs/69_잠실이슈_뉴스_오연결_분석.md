# 잠실 스포츠 MICE 파크 이슈 - 뉴스 오연결 분석

날짜: 2026-03-11
상태: ❌ 심각한 오연결 문제

## 문제

**이슈:**
```
'3만석 돔구장·코엑스 2.5배 전시'…잠실 '스포츠·MICE 파크' 사업 본격
```

**연결된 뉴스: 111건**

**문제점:**
- 대부분 **"스포츠" 단어 하나만 공통**인 전혀 관련 없는 뉴스
- 실제 잠실 돔구장/MICE 파크 관련 뉴스는 0건

**예시 (오연결 뉴스):**
```
1. 웨이브, KLPGA 및 KPGA 전 경기 온라인 생중계하며 스포츠 콘텐츠 강화
   공통 토큰: [스포츠] (1개)

2. 스포츠토토 건전화 통합 플랫폼 '위드토토' 오픈 기념 이벤트 실시
   공통 토큰: [] (0개)

3. 무신사, 스포츠·IP 전문 큐레이션관 '팬 스토어' 론칭
   공통 토큰: [스포츠] (1개)

4. 용인시, 경기도내 최초 생활체육 AI 스포츠 중계 시스템 도입
   공통 토큰: [스포츠] (1개)
```

## 원인 분석

### 1. 이슈 토큰 확인

```typescript
이슈 제목: '3만석 돔구장·코엑스 2.5배 전시'…잠실 '스포츠·MICE 파크' 사업 본격

토큰화 결과: [3만석, 돔구장, 코엑스, 5배, 전시, 잠실, 스포츠, MICE, 파크, 사업, 본격]
토큰 수: 11개
```

### 2. 뉴스 연결 기준

**`lib/linker/issue-news-linker.ts` 로직:**

```typescript
// 임계값 계산
const threshold = Math.min(5, Math.max(3, Math.ceil(keywordsLower.length * 0.7)))

// 11개 키워드 → 70% = 7.7 → ceil(7.7) = 8
// min(5, max(3, 8)) = 5

실제 적용된 임계값: 5개
```

**문제:**
- 이슈 토큰 11개
- 임계값 5개 (45%)
- 카테고리 필터: "사회"

### 3. 오연결 발생 원인

**시나리오:**

1. 이슈 카테고리: "사회"
2. 뉴스 카테고리: "사회" or null
3. 뉴스 제목: "웨이브, KLPGA 및 KPGA 전 경기 온라인 생중계하며 **스포츠** 콘텐츠 강화"
4. 공통 토큰: ["스포츠"] = 1개

**어떻게 연결되었나?**

```typescript
// linker-utils.ts - isMatch 함수
export function isMatch(title: string, keywordsLower: string[], threshold: number): boolean {
    const titleWords = extractTitleWords(title)
    const matchCount = keywordsLower.filter((kw) => titleWords.has(kw)).length
    return matchCount >= threshold
}
```

**실제 매칭 과정:**

```
이슈 키워드: [3만석, 돔구장, 코엑스, 5배, 전시, 잠실, 스포츠, mice, 파크, 사업, 본격]

뉴스 제목: "웨이브, KLPGA 및 KPGA 전 경기 온라인 생중계하며 스포츠 콘텐츠 강화"
뉴스 단어: [웨이브, klpga, 및, kpga, 전, 경기, 온라인, 생중계하며, 스포츠, 콘텐츠, 강화]

공통 단어: [스포츠] = 1개

1개 >= 5개? → NO → 연결 안 됨 (정상)
```

**그런데 왜 연결되었나?**

가능한 원인:
1. **이슈 생성 시점의 키워드가 다름** (keywords 필드 vs title 토큰)
2. **카테고리가 null인 뉴스** (카테고리 필터 우회)
3. **날짜 범위 문제** (전후 3일 설정)

### 4. 실제 이슈 데이터 확인

```sql
이슈 키워드 필드: []  ← 빈 배열!
```

**문제 발견!**

이슈의 `keywords` 필드가 **빈 배열**입니다. 이 경우:

```typescript
// issue-news-linker.ts:53
const keywords = extractKeywords(issueTitle)
if (keywords.length === 0) return 0  // 빈 배열이면 종료

// 하지만 extractKeywords는 빈 배열을 반환하지 않음
```

실제 확인 필요: 이슈 생성 당시 어떤 키워드가 사용되었는지

### 5. 뉴스 카테고리 확인

```typescript
// 카테고리 필터 로직
.or(`category.eq.${issueCategory},category.is.null`)
```

**문제:**
- 카테고리가 null인 구 뉴스도 매칭 대상
- "스포츠" 키워드만으로 연결 가능

## 근본 원인

### 원인 1: 범용 키워드 문제

**"스포츠" 키워드는 너무 범용적**

```
이슈 제목에 "스포츠"가 포함되면:
- 스포츠 중계
- 스포츠 마케팅
- 스포츠 토토
- 스포츠 브랜드
- 생활 스포츠
등 모든 스포츠 관련 뉴스가 매칭 가능
```

### 원인 2: 특수 키워드 필터 부족

```typescript
// linker-utils.ts - STOPWORDS
const STOPWORDS = new Set([
    '논란', '사건', '사고', ...
    // "스포츠"는 포함 안 됨
])
```

**"스포츠"를 불용어로 추가할 수 없는 이유:**
- 진짜 스포츠 이슈도 많음 (선수, 팀, 경기 등)
- "스포츠·MICE 파크"처럼 중요한 키워드일 수 있음

### 원인 3: 복합어 처리 부족

```
"스포츠·MICE 파크" → 토큰화 → [스포츠, MICE, 파크]

단독 "스포츠"가 아닌 "스포츠MICE파크" 또는 "스포츠파크"로 처리해야
다른 스포츠 뉴스와 구분 가능
```

## 해결 방안

### Option 1: 복합 키워드 우선 처리 (권장)

**개선: `lib/linker/linker-utils.ts`**

```typescript
// 복합 키워드 패턴 정의 (우선 추출)
const COMPOUND_PATTERNS = [
    /스포츠[\s·]+MICE[\s·]+파크/gi,
    /MICE[\s·]+파크/gi,
    /돔[\s·]+구장/gi,
    // 추가 패턴...
]

export function extractKeywords(text: string): string[] {
    const cleaned = stripMediaPrefix(text)
    
    // 1. 복합 키워드 먼저 추출
    const compounds: string[] = []
    for (const pattern of COMPOUND_PATTERNS) {
        const matches = cleaned.match(pattern)
        if (matches) {
            compounds.push(...matches.map(m => m.replace(/[\s·]+/g, '')))
        }
    }
    
    // 2. 일반 키워드 추출
    const words = cleaned
        .replace(/[^\w\sㄱ-ㅎㅏ-ㅣ가-힣]/g, ' ')
        .split(/\s+/)
        .filter((w) => (w.length >= 2 || ALLOWED_ONE_CHAR_KEYWORDS.has(w)) && !STOPWORDS.has(w))
    
    // 3. 복합 키워드 + 일반 키워드
    return Array.from(new Set([...compounds, ...words]))
}
```

**효과:**
```
Before: [3만석, 돔구장, 코엑스, 5배, 전시, 잠실, 스포츠, MICE, 파크, 사업, 본격]
After:  [스포츠MICE파크, 돔구장, 3만석, 코엑스, 5배, 전시, 잠실, 사업, 본격]

"스포츠" 단독 키워드는 제외되고, "스포츠MICE파크" 복합어만 매칭
```

### Option 2: 범용 키워드 가중치 낮춤

```typescript
const GENERIC_KEYWORDS = ['스포츠', '문화', '행사', '사업', '개최']

export function isMatch(title: string, keywordsLower: string[], threshold: number): boolean {
    const titleWords = extractTitleWords(title)
    
    // 일반 키워드는 0.5점, 특수 키워드는 1점
    let matchScore = 0
    for (const kw of keywordsLower) {
        if (titleWords.has(kw)) {
            matchScore += GENERIC_KEYWORDS.includes(kw) ? 0.5 : 1
        }
    }
    
    return matchScore >= threshold
}
```
