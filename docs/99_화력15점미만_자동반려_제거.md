# 화력 15점 미만 자동 반려 제거

날짜: 2026-03-16

## 문제 인식

관리자 이슈 관리 페이지에 화력 15점 미만 이슈가 노출되는 문제가 있었습니다.

## 원인 분석

### 불일치하는 처리 로직

**트랙 A 등록 시** (정상):
- 화력 15점 미만이면 이슈를 즉시 삭제
- 등록 자체가 되지 않음

**화력 재계산 시** (문제):
- 화력이 15점 미만으로 하락하면 "반려" 상태로 변경
- 이슈는 삭제하지 않음
- 관리자 페이지에 계속 노출

### 핵심 문제점

한 번 15점 이상으로 등록된 이슈는 이미 의미 있다고 판단된 것입니다.
시간 가중치로 화력이 떨어졌다고 해서 자동으로 반려하는 것은 부적절합니다.

**이유:**
- 이슈는 시간이 지나면 자연스럽게 화력이 떨어짐 (시간 가중치 적용)
- 생성 시점의 `created_heat_index`가 15점 이상이었으면 유효한 이슈
- 오래된 이슈도 관리자가 검토하여 승인 여부 판단 필요

## 해결 방안

### 자동 반려 로직 제거

화력 재계산 시 화력이 15점 미만으로 떨어져도:
- **자동 반려 안 함**
- **대기 상태 유지**
- 관리자가 직접 검토하여 승인/반려 결정

### 수정 파일

#### 1. 화력 재계산 Cron

파일: `app/api/cron/recalculate-heat/route.ts`

**Before:**
```typescript

    // 자동 승인
} else if (heatIndex < MIN_HEAT_TO_REGISTER && !isNewIssue) {
    // 자동 반려: 화력 미달 (단, 생성 10분 이내 이슈는 제외)
    await supabaseAdmin
        .from('issues')
        .update({ 
            approval_status: '반려',
            approval_type: 'auto',
        })
        .eq('id', issue.id)
}
```

**After:**
```typescript
if (shouldAutoApprove(category, heatIndex)) {
    // 자동 승인
}
// 자동 반려 제거:
// - 화력이 15점 미만으로 떨어져도 자동 반려하지 않음
// - 이미 등록된 이슈는 관리자가 직접 판단
// - 화력 15-29점 → 대기 유지 (관리자 승인 필요)
```

**After:**

### 이슈 등록 기준 (변경 없음)

트랙 A에서 이슈 등록 시:
- 화력 15점 이상만 등록
- 15점 미만이면 이슈 삭제 (등록 안 함)

### 화력 재계산 정책 (변경됨)

10분마다 화력 재계산 시:

**자동 승인:**
- 화력 30점 이상 + 허용 카테고리 (사회/기술/스포츠)
- `approval_status`: 대기 → 승인
- `approval_type`: auto

**대기 유지:**
- 화력 15-29점 (모든 카테고리)
- 화력 30점 이상 (연예/정치)
- **화력 15점 미만 (NEW: 자동 반려 안 함)**
- `approval_status`: 대기 유지
- 관리자 승인 필요

**자동 반려:** (제거됨)

## 관리자 작업 영향

### 기존 문제

- 화력 15점 미만으로 자동 반려된 이슈가 목록에 노출
- 의미 없는 반려 이슈 확인 필요

### 개선 효과

- 화력 15점 미만 이슈도 대기 상태 유지
- 관리자가 화력 추이와 내용을 보고 판단
- 시간이 지나 화력이 떨어진 유효한 이슈도 승인 가능

### 관리자 판단 기준

화력이 낮은 이슈 검토 시:
1. `created_heat_index` 확인 (등록 시점 화력)
2. 화력 추이 확인 (등록 시 → 현재)
3. 이슈 내용과 타임라인 검토
4. 연결된 뉴스/커뮤니티 데이터 확인
5. 최종 승인/반려 결정

## 데이터 정리

### 화력 15점 미만 이슈 삭제 (2026-03-16)

**삭제된 이슈:**
- 총 29개 이슈 삭제
  - 승인 상태: 6개
  - 반려 상태: 23개

**삭제 사유:**
- 화력 15점 미만으로 등록 기준 미달
- 시간 가중치로 화력이 0-14점으로 하락
- 대부분 오래된 이슈 또는 관련성이 떨어지는 이슈

**주요 삭제 이슈:**
- WBC 관련 오래된 이슈 (화력 0-3점)
- 해외 뉴스 (이탈리아, 멕시코 등)
- 연예 뉴스 (화력 1-3점)
- 정치 이슈 (화력 0-1점)

관리자가 검토하여:
- 유효한 이슈 → 승인
- 무의미한 이슈 → 반려 유지 또는 삭제

## 장점

1. **일관성**: 트랙 A는 등록 안 함, 재계산은 대기 유지
2. **유연성**: 관리자가 맥락을 보고 판단
3. **정확성**: 시간 가중치 영향을 고려한 판단
4. **안전성**: 유효한 이슈를 자동으로 반려하지 않음

## 결론

화력 15점은 **등록 기준**이지 **유지 기준**이 아닙니다.

- **등록 시**: 화력 15점 이상 필요
- **등록 후**: 화력이 떨어져도 대기 상태 유지
- **최종 결정**: 관리자 판단
