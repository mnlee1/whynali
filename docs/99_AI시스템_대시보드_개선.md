# AI 시스템 대시보드 변경

## 변경 내용

Perplexity API 섹션을 제거하고, 현재 실제로 사용 중인 AI 시스템 현황을 표시하도록 변경했습니다.

## 기존 (변경 전)

- Perplexity AI API 상세 정보 표시
  - 토큰 사용량 (입력/출력)
  - 호출 횟수
  - 비용
- 문제점: **실제로는 사용하지 않음** (ENABLE_AI_GROUPING=false)

## 개선 (변경 후)

### 현재 사용 중인 AI 표시

**Groq AI (Llama 3.1) - 무료**
- 실시간 상태 표시 (활성/비활성)
- 사용 중인 기능 목록:
  - 커뮤니티 매칭
  - 중복 이슈 체크
  - 토론 주제 생성
  - 투표 생성
- 통계:
  - 오늘/월별 호출 횟수
  - 성공률 (%)

### UI 개선

1. **상태 표시**
   - 녹색 점 애니메이션으로 활성 상태 표시
   - "사용 중" 배지 표시

2. **기능별 상태**
   - 각 AI 기능의 활성화 여부를 한눈에 확인
   - 녹색 배지로 활성 기능 강조

3. **성공률 표시**
   - AI 호출 성공률을 백분율로 표시
   - 오류 발생 시 빠르게 파악 가능

## 변경된 파일

### 프론트엔드
- `app/admin/page.tsx`: UI 전면 개편
  - Perplexity 섹션 제거
  - Groq AI 섹션 추가
  - 타입 인터페이스 변경

### 백엔드
- `lib/api-usage-tracker.ts`: API 응답 변경
  - `getAllApiCostsSummary()` 함수 수정
  - Perplexity 데이터 제거
  - Groq 데이터 추가 (호출 횟수, 성공/실패 건수)

## 환경변수 확인

현재 활성화된 AI 기능:

```bash
# Groq AI 사용 (무료)
ENABLE_AI_COMMUNITY_MATCHING=true
ENABLE_AI_DUPLICATE_CHECK=true

# Perplexity AI 비활성화
ENABLE_AI_GROUPING=false
ENABLE_AI_GROUP_VALIDATION=false
```

1. **정확한 정보**: 실제로 사용 중인 AI만 표시

## 향후 계획

1. `.env.local`에서 `ENABLE_AI_GROUPING=true` 설정
2. 대시보드에 Perplexity 섹션 자동 표시 (조건부 렌더링 추가 필요)
