상태: ✅ 해결 완료

## 문제

같은 이슈인데 묶이지 않음:

```
1. 유한재단, 사회보장정보원과 돌봄 청소년·청년 지원 업무협약 체결
2. 유한재단, 한국사회보장정보원과 맞손…돌봄 청소년·청년 지원사업 추
```

## 원인 분석

### 토크나이징 문제

**Before (수정 전):**
```
제목 1: [유한재단, 사회보장정보원과, 돌봄, 청소년, 청년, 업무협약, 체결]
제목 2: [유한재단, 한국사회보장정보원과, 맞손, 돌봄, 청소년, 청년, 지원사업]

공통 토큰: 4개 (유한재단, 돌봄, 청소년, 청년)
```

**문제점:**
1. "사회보장정보원**과**" - 조사 "과"가 붙어서 다른 토큰
2. "한국사회보장정보원" vs "사회보장정보원" - 기관명 앞 "한국" 미정규화

### 그루핑 조건

```typescript
GROUPING_MIN_COMMON: 2 (최소 공통 키워드)
실제 공통 키워드: 4개

✅ 그루핑 조건은 충족
```

**그런데 왜 안 묶였나?**

이 두 이슈는 **다른 시간에 수집된 별도 cron 실행**에서 생성되었습니다:
- 1번 이슈: 2026-03-11 10:51:24 (첫 번째 수집)
- 2번 이슈: 2026-03-11 13:29:17 (두 번째 수집)

같은 배치 내에서는 그루핑되지만, **다른 배치 간 중복 체크는 없었습니다**.

## 해결 방법

### 1. 토크나이저 개선

**수정 내역: `lib/candidate/tokenizer.ts`**

```typescript
export function tokenize(text: string): string[] {
    const words = text
        .replace(/[^\wㄱ-ㅎㅏ-ㅣ가-힣\s]/g, ' ')
        .split(/\s+/)
        .map((w) => {
            let normalized = w
            
            // 1. "XXX와", "XXX과" 형태의 조사 제거
            if (normalized.endsWith('와') || normalized.endsWith('과')) {
                normalized = normalized.slice(0, -1)
            }
            
            // 2. "한국XXX" → "XXX" 정규화 (기관명, 4글자 이상만)
            if (normalized.startsWith('한국') && normalized.length > 4) {
                normalized = normalized.slice(2)
            }
            
            return normalized
        })
        .filter((w) => (w.length >= 2 || ALLOWED_ONE_CHAR_KEYWORDS.has(w)) && !GROUPING_STOPWORDS.has(w))
        .map(normalizeKeyword)
    return Array.from(new Set(words))
}
```

### 추가 개선 필요 (선택)

현재는 같은 cron 실행 내에서만 그루핑됩니다. 배치 간 중복을 방지하려면:

**Option 1: 이슈 생성 전 유사도 체크**
```typescript
// lib/candidate/issue-candidate.ts
async function checkExistingIssues(title: string) {
    const tokens = tokenize(title)
    
    // 최근 7일 이슈 조회
    const { data: recentIssues } = await supabase
        .from('issues')
        .select('id, title')
        .gte('created_at', sevenDaysAgo)
    
    // 토큰 유사도 체크
    for (const issue of recentIssues) {
        const issueTokens = tokenize(issue.title)
        const common = commonKeywordCount(tokens, issueTokens)
        
        if (common >= 5) {
            // 기존 이슈에 추가
            return issue.id
        }
    }
    
    // 새 이슈 생성
    return null
}
```
