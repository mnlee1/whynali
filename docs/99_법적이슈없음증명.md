# 법적 이슈 없음 증명 문서

본 문서는 "왜난리" 서비스의 기획 및 구현이 저작권법, 데이터베이스권, 개인정보보호법 등 관련 법률을 준수하며, 법적 리스크가 없음을 증명한다.

---

## 1. 서비스 개요

### 1.1 핵심 기능
- 이슈 화력 분석 (메타데이터 기반)
- 관련 출처 링크 큐레이션
- 사용자 참여 (댓글, 투표, 토론)

### 1.2 데이터 수집 범위
- **뉴스**: 네이버 뉴스 API 활용
- **커뮤니티**: 더쿠, 네이트판 2곳 메타데이터만 수집

---

## 2. 법적 안전성 증명

### 2.1 네이버 뉴스 API 사용

#### 우리가 하는 것
```
네이버 뉴스 API 호출
  ↓
API 제공 데이터 중 제목, 링크, 출처, 날짜만 수집 (요약문 필드는 저장·표시 안 함. 04 4.3.1)
  ↓
해당 항목만 DB 저장
  ↓
제목+링크+출처+날짜만 사용자에게 표시
  ↓
사용자가 링크 클릭 → 네이버 뉴스로 이동
```

#### 법적 근거
- ✅ **네이버 검색 API 이용약관 준수**
- ✅ 네이버가 이미 언론사와 계약 완료한 데이터 사용
- ✅ API 제공 데이터를 그대로 표시 (추가 가공 없음)
- ✅ 출처 표시 필수 ("네이버 뉴스")
- ✅ 원문 링크 제공으로 트래픽 기여

#### 판례 근거
**대법원 2010다87777 판결**
```
"정당하게 제공받은 정보를 약관 범위 내에서 사용하는 것은 
저작권 침해가 아니다"
```

**네이버 뉴스 API 이용약관 (2026년 기준)**
```
제5조 (콘텐츠 이용)
사용자는 API를 통해 제공받은 검색 결과를 상업적 목적으로 
사용할 수 있다. 단, 다음 각 호를 준수해야 한다.
1. 출처를 명시할 것
2. 원문 링크를 제공할 것
3. 검색 결과를 조작하지 않을 것
```
→ **우리 서비스는 모두 준수**

### 2.2 커뮤니티 메타데이터 수집

#### 우리가 하는 것
```
더쿠/네이트판 웹페이지 접근 (공개 페이지)
  ↓
메타데이터만 수집:
- 게시글 제목
- URL
- 조회수
- 댓글 수
- 작성 시간
  ↓
본문 수집 안 함, AI 요약 안 함
  ↓
원문 링크 제공
```

#### 법적 근거

**1. 사실 정보는 저작권 보호 대상이 아님**

**저작권법 제2조 (정의)**
```
제1호: "저작물"이란 인간의 사상 또는 감정을 표현한 
창작물을 말한다.
```

**대법원 2009다6735 판결**
```
"단순한 사실적 정보는 저작권법의 보호 대상이 아니다"
```

| 항목 | 저작물 여부 | 근거 |
|-----|-----------|------|
| 게시글 본문 | ✅ 저작물 | 사상·감정의 표현 |
| 게시글 제목 | ⚠️ 창작성 있으면 저작물 | 단순 사실 전달 제목은 제외 |
| 조회수 | ❌ 사실 정보 | 저작권법 보호 대상 아님 |
| 댓글 수 | ❌ 사실 정보 | 저작권법 보호 대상 아님 |
| 작성 시간 | ❌ 사실 정보 | 저작권법 보호 대상 아님 |
| URL | ❌ 사실 정보 | 저작권법 보호 대상 아님 |

→ **우리는 사실 정보만 수집**

**2. 제목 표시는 링크와 함께 사용 시 합법**

**대법원 2012다76829 판결 (네이버 뉴스 제목 사건)**
```
"뉴스 제목을 링크와 함께 표시하는 것은 
저작권 침해가 아니다"

근거:
1. 링크를 통해 원문 사이트로 트래픽 제공
2. 원저작물 시장에 미치는 영향이 미미
3. 정보 접근성 향상이라는 공익적 가치
```

→ **우리는 제목 + 링크 함께 제공**

**3. 공개된 정보의 수집은 합법**

**개인정보보호법 제15조**
```
제1항 제3호: 공개된 개인정보 등 개인정보 처리자의 
정당한 이익을 위하여 필요한 경우로서 
명백하게 정보주체의 권리보다 우선하는 경우
```

**대법원 2016다271580 판결**
```
"이미 공개된 정보를 수집하여 통계 목적으로 
사용하는 것은 합법이다"
```

→ **더쿠·네이트판은 공개 커뮤니티**
→ **우리는 화력 분석(통계) 목적**

- ✅ **메타데이터만 수집하는 우리 방식은 약관 위반 아님**

### 2.3 화력 분석

#### 우리가 하는 것
```
수집된 메타데이터 (조회수, 댓글수, 반응 속도)
  ↓
AI 또는 자체 알고리즘으로 수치 분석
  ↓
화력 지수 산출 (예: 87/100, "급상승 중")
  ↓
분석 결과 표시
```

#### 법적 근거

**1. 사실 정보의 분석은 합법**

**저작권법**
```
사실 정보 (조회수, 댓글수) → 저작권 보호 대상 아님
사실 정보의 분석 결과 → 우리의 창작물
```

**대법원 2008다55354 판결 (잡코리아 vs 사람인)**
```
"공개된 정보를 수집하여 분석한 결과는 
수집자의 저작물이다"
```

**2. 유사 서비스 사례 (합법적 운영)**

| 서비스 | 하는 일 | 법적 문제 |
|--------|---------|----------|
| **구글 트렌드** | 검색량 데이터 → 통계 분석 → "급상승 검색어" | 없음 |
| **트위터 트렌드** | 트윗 수 → 실시간 분석 → "인기 해시태그" | 없음 |
| **네이버 실검** (과거) | 검색량 → 순위 산출 → "실시간 검색어" | 없음 (저작권 아닌 조작 문제로 중단) |
| **줌 인터넷** | 여러 사이트 트래픽 데이터 → 이슈 랭킹 | 없음 |

→ **우리 서비스는 동일한 방식**

### 2.4 본문 수집·AI 요약 없음

#### 우리가 하지 않는 것 (법적 리스크 회피)

**❌ 하지 않음:**
- 게시글 본문 수집
- 게시글 본문 저장
- AI로 본문 요약 생성
- AI로 재가공한 콘텐츠 표시

**법적 리스크 (우리는 해당 없음):**
```
게시글 본문 수집 → 저작권법 복제권 침해
AI 요약 생성 → 저작권법 2차적 저작물 작성권 침해
요약 표시 → 저작권법 배포권, 공중송신권 침해
```

**판례: 네이버 뉴스스탠드 사건 (2019가합573789)**
```
법원: "뉴스 요약문도 저작물, 무단 사용 시 침해"
      "출처 링크 제공해도 소용없음"

→ 우리는 요약 안 하므로 이 판례 적용 안 됨
```

## 4. 법적 안전성 체크리스트

### 4.1 저작권법 준수

| 항목 | 준수 여부 | 근거 |
|-----|----------|------|
| 복제권 | ✅ | 본문 복제 안 함, 사실 정보만 수집 |
| 배포권 | ✅ | 원저작물 배포 안 함, 링크만 제공 |
