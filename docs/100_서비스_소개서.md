# 왜난리(WhyNali) 서비스 소개서

> 한국 이슈를 한눈에 파악하고 여론을 확인하는 이슈 내비게이션 서비스

---

## 서비스 개요

**왜난리(WhyNali)**는 뉴스·커뮤니티에 분산된 한국 이슈 정보를
한 곳에 모아 시계열로 정리해 제공하는 이슈 내비게이션 서비스입니다.

- URL: [whynali.com](https://whynali.com)
- 플랫폼: 반응형 웹 (모바일 퍼스트, PC 지원)
- 로그인: 구글 / 네이버 / 카카오 오픈ID

---

## 문제 정의

연예·스포츠·정치·사회·기술 등 주요 이슈 정보는
뉴스, 커뮤니티, SNS에 분산되어 있습니다.

- "지금 무슨 일이 벌어졌는지"를 한번에 파악하기 어렵다
- 이슈가 왜 커졌는지, 어떻게 흘러갔는지 흐름을 알기 어렵다
- 이슈의 진행 과정을 시계열로 정리해주는 서비스가 없다
- 탐색 비용이 높아 재방문 동기가 약하다

## 주요 기능

### 타임라인

이슈의 흐름을 4단계로 시각화합니다.

```
발단 → 전개 → 파생 → 진정
```

- 각 포인트에 출처 링크 연결
- 시간순 정렬, 세로 스크롤

### 화력 지수

이슈가 얼마나 뜨겁게 타오르는지
메타데이터 기반으로 산출합니다.

- 범위: 0~100 (정수)
- 등급: 높음(70 이상) / 보통(30~69) / 낮음(29 이하)
- 자동 계산, 관리자 승인 불필요

### 이슈 상태

| 상태 | 의미 |
|------|------|
| 점화 | 막 시작되어 확산 중 |
| 논란중 | 양측 의견 대립 중 |
| 종결 | 사과문 발표 또는 결론 도출 |

### 감정 표현 7종

좋아요 / 싫어요 / 화나요 / 팝콘각 / 응원 / 애도 / 사이다

- 이슈당 1인 1회, 변경·취소 가능
- 실시간 집계 표시

### 투표

이슈별 복수 투표를 지원합니다.
시점·상황별 여론 변화를 시계열로 추적할 수 있습니다.

- 선택지 2~6개 지원
- 실시간 득표 비율 그래프
- 자동 마감: 날짜 기준 / 참여자 수 달성 / 이슈 종결 연동

### 댓글

- 베스트 댓글 상단 노출 (좋아요 - 싫어요 기준)
- 정렬: 최신순 / 좋아요순 / 싫어요순
- 대댓글 지원

### 커뮤니티 (토론)

이슈에서 파생된 토론 주제에
철학적 관점으로 의견을 남기는 공간입니다.

- AI가 이슈 메타데이터 기반으로 토론 주제 자동 생성
- 관리자 승인 후 등록
- 유저는 댓글로 참여 (로그인 필요)

## 데이터 수집 방식

### 뉴스

- 출처: 네이버 뉴스 API (공식 라이선스)
- 수집 주기: 30분
- 수집 항목: 제목, 링크, 출처, 날짜만
- 제외: 기사 본문, AI 요약

### 커뮤니티

- 출처: 더쿠, 네이트판
- 수집 주기: 3분
- 수집 항목: 제목, URL, 조회수, 댓글수, 작성시간 (메타데이터만)
- 제외: 게시글 본문, 닉네임

---

## 이슈 자동화 흐름

```
수집 (뉴스 30분 / 커뮤니티 3분)
    ↓
이슈 후보 자동 생성 (동일 이슈 5건 이상)
    ↓
AI 카테고리 분류 + 중복 체크
    ↓
화력 지수 계산
    ↓
관리자 승인 또는 자동 승인
    ↓
이슈 공개 (타임라인 + 출처 + 감정/댓글/투표)
```
