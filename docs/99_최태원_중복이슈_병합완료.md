# 최태원 회장 중복 이슈 병합 완료

## 문제 상황
최태원 회장 관련 이슈가 6개로 쪼개져 있었음

### 중복 발생 원인
1. **AI 중복 그룹 체크 비활성화**
   - `ENABLE_AI_DUPLICATE_GROUP_CHECK=false`로 설정되어 있었음
   - 키워드 그루핑만으로는 표현이 다른 같은 이슈를 묶지 못함

2. **키워드 그루핑 한계**
   - 공통 키워드가 "최태원" 1개뿐
   - 나머지 키워드: "윤호중", "사회문제", "경제성장", "사회적비용", "GDP", "AI" 등 모두 다름
   - 그루핑 조건 미달: `공통 2개 이상` 또는 `공통 1개 + 같은 카테고리`

3. **시간차 수집 문제**
   - 오전 11시~오후 3시: 같은 이벤트의 뉴스가 여러 시간대에 수집됨
   - 각 시간대마다 별개 이슈로 생성됨

## 해결 방안

### 1. 환경변수 수정 (향후 중복 방지)
```env
# Before
ENABLE_AI_DUPLICATE_GROUP_CHECK=false

# After
ENABLE_AI_DUPLICATE_GROUP_CHECK=true
```

이제 키워드 그루핑 후 AI가 한번 더 검증하여 같은 이슈면 자동 병합합니다.

### 2. 기존 중복 이슈 병합 완료

**병합 전:**
```
이슈 1: "사회적 가치 기반 성장" 공감대 (화력 20점, 뉴스 11건) - 오전 11시
이슈 2: "사회적 비용 줄이면 경제 이득" (화력 25점, 뉴스 1건) - 오후 1시 30분
이슈 3: "경제 성장, 사회 문제 해결" (화력 23점, 뉴스 2건) - 오후 1시 36분
이슈 4: "AI 활용 사회적 기업" (화력 1점, 뉴스 1건) - 오후 2시 40분
이슈 5: "사회적 비용 줄이는 게 새 성장동력" (화력 27점, 뉴스 1건) - 오후 2시 51분
이슈 6: "최태원·윤호중 사회문제 해결" (화력 15점, 뉴스 14건) - 오후 3시 41분
```

**병합 후:**
```
통합 이슈: "최태원·윤호중 사회문제 해결과 경제성장 함께"
- ID: 2a8324e1-31cb-4837-bf43-ce78fb14da35
- 카테고리: 사회 (연예에서 수정)
- 총 화력: 111점 (6개 합산)
- 연결 뉴스: 30건 (모든 뉴스 통합)
- 상태: 대기
```

**병합 작업 내역:**
1. ✅ news_data 30건 → Primary 이슈로 이동
2. ✅ community_data → Primary 이슈로 이동
3. ✅ timeline_points → Primary 이슈로 이동
4. ✅ emotional_expressions → Primary 이슈로 이동
5. ✅ issue_comments → Primary 이슈로 이동
6. ✅ issue_votes → Primary 이슈로 이동
7. ✅ Secondary 이슈 5개 → `approval_status='병합됨'` + `merged_into_id` 설정
8. ✅ Primary 이슈 화력 재계산: 111점
9. ✅ 카테고리 수정: 연예 → 사회

## 최종 결과

- 6개 이슈 → 1개 통합 완료
- 모든 데이터(뉴스 30건) 하나의 이슈에 통합
- 병합된 5개 이슈는 목록에서 제외 (UI에서 안 보임)
- 향후 같은 문제 발생 방지 (AI 중복 체크 활성화)

## 파일 변경
- `.env.local`: `ENABLE_AI_DUPLICATE_GROUP_CHECK=true`
- `scripts/merge-choi-taewon-issues.mjs`: 병합 스크립트 (1회성, 재사용 가능)

## 참고
병합된 이슈는 DB에서 삭제하지 않고 이력으로 보관합니다.
필요시 `scripts/check-consistency.mjs`로 병합 데이터 정합성을 점검할 수 있습니다.
