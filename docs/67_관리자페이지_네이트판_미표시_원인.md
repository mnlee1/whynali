# 관리자 페이지 네이트판 미표시 원인 분석

날짜: 2026-03-11
상태: ✅ 데이터 정상, UI 표시 문제

## 원인

### 데이터 수집 상태

```
✅ 더쿠: 881건 (평균 댓글 15.5개)
✅ 네이트판: 119건 (평균 댓글 9.7개)
```

**데이터는 정상 수집 중**

### 관리자 페이지 문제

**기본 정렬: 댓글수 내림차순**

```
TOP 20 (댓글순):
- 더쿠: 20건
- 네이트판: 0건

이유:
- 더쿠 최대 댓글수: 1,302개
- 네이트판 최대 댓글수: 204개
- 네이트판이 더쿠보다 댓글수가 적어서 TOP 20 밖으로 밀림
```

### 댓글수 비교

**더쿠 TOP 5:**
```
1. 왜 다들 가정사가 있어도... (댓글 1,302)
2. 셋 다 모를 수는 있어도... (댓글 1,228)
3. 부산 해운대로 이사와서... (댓글 1,200)
4. 민희진 풋옵션 256억... (댓글 1,179)

```

**네이트판 TOP 5:**
```

2. 엔하이픈 희승 탈퇴 (댓글 177)

```

## 해결 방법

### 사용자 액션 (현재)

**방법 1: 네이트판 탭 클릭**
```
관리자 페이지 → 커뮤니티 섹션 → "네이트판" 탭 클릭
→ 네이트판만 필터링되어 보임
```

**방법 2: 정렬 변경**
```
"댓글" 헤더 클릭 → "작성일" 또는 "수집일" 헤더 클릭
→ 시간순으로 정렬되면 네이트판도 보임
```

### UI 개선 (선택)

#### Option 1: 기본 정렬 변경 (권장)

**현재:**
```typescript
const [communitySort, setCommunitySort] = useState<CommunitySort>('comment_count')
```

**제안:**
```typescript
const [communitySort, setCommunitySort] = useState<CommunitySort>('created_at')
```

**효과:**
- 최신 수집 데이터부터 보임
- 사이트 균형있게 표시
- 수집 상태 모니터링에 적합

**단점:**
- 인기글 우선 확인 불가

#### Option 2: 사이트별 교차 정렬

**제안:**
```typescript
// 더쿠 10개 + 네이트판 10개 교차 표시
// 또는 사이트별 비율 유지 (더쿠 15개 + 네이트판 5개)
```

**효과:**
- 두 사이트 골고루 표시
- 수집 상태 한눈에 파악

**단점:**
- 구현 복잡도 증가
- 정렬 의미 모호

#### Option 3: 현재 상태 유지 + 안내 추가

**제안:**
```typescript
// 커뮤니티 섹션 상단에 힌트 추가
{communitySiteFilter === '전체' && communitySort === 'comment_count' && (

    </div>
)}
```

**효과:**
- 사용자 혼란 방지
- 코드 변경 최소화

## 권장 사항

### 즉시 적용 (Option 1 권장)

**기본 정렬을 `created_at`으로 변경**

**이유:**
1. 관리자 페이지 목적 = 수집 모니터링
2. 최신 데이터 확인이 더 중요
3. 사이트별 수집 상태 균형있게 파악
4. 인기글은 별도 페이지에서 확인 가능

**변경 위치:**
```
whynali/app/admin/collections/page.tsx
Line 226: const [communitySort, setCommunitySort] = useState<CommunitySort>('created_at')
```

### 장기 개선 (Option 2)

**사이트별 균형 표시**
- 더쿠/네이트판 수집량 비율만큼 표시
- 또는 고정 비율 (예: 15:5)

## 결론

**문제:**
- 네이트판 데이터는 정상 수집 중
- 기본 정렬(댓글순)에서 더쿠가 압도적이라 네이트판이 안 보임

**해결:**
- 기본 정렬을 `created_at`으로 변경 (1줄 수정)
- 또는 사용자가 "네이트판" 탭 클릭

**권장:**
- **즉시 적용**: 기본 정렬 변경
- **효과**: 수집 모니터링 목적에 적합, 사이트 균형 표시
