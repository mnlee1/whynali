## 요약

**궁극적 목표 (Why)**
왜난리를 "한국 이슈를 빠르게 파악하고 반응할 수 있는 커뮤니티"로 자리잡게 해서, 2026년 안에 외부 유입 기반의 실유저 880명을 확보한다.

**전략 방향**
- Phase 1 (6~8월): 실유저 확보 — SNS 카드뉴스로 유입, 이슈 자동 등록 시스템으로 방문 이유 제공
- Phase 2 (9~12월): 참여 & 재방문 — 실제 데이터 기반으로 콘텐츠·화면 개선

## SMART KPI 7개

---

### KPI 1 — 월간 신규 가입자

- **S (구체적):** Instagram·Threads·X 카드뉴스 링크를 통해 유입된 외부 회원의 신규 가입 수. 운영자 3명 제외.
- **M (측정 방법):** 매월 새로 가입한 회원 수 집계
- **A (달성 근거):** 3채널 × 일 1건 카드뉴스로 팔로워 누적 효과 발생. 광고 없이 SNS 꾸준한 운영만으로 달성 가능한 수치.
- **R (연관성):** 실유저가 없으면 참여율·재방문율 등 모든 지표가 의미 없음. 가장 우선순위 높은 지표.
- **T (기한):** 매월 말일 측정

> **최종 KPI:** Instagram·Threads·X 카드뉴스 운영을 통해 2026년 12월 31일까지 외부 유입 기반 실유저 누적 880명을 달성한다. (현재 19명 → 6월 65명 → 12월 880명)

| 월 | 목표 누적 가입자 | 월 신규 목표 |
|----|--------------|------------|
| 6월 | **65명** | +43명 |
| 7월 | **115명** | +50명 |
| 8월 | **195명** | +80명 |
| 9월 | **310명** | +115명 |
| 10월 | **465명** | +155명 |
| 11월 | **645명** | +180명 |
| 12월 | **880명** | +235명 |

### KPI 2 — 일간 이슈 승인 건수

- **S (구체적):** 하루 기준 승인 처리된 이슈 수. 자동승인(화력 기준 초과 시 자동 처리)과 관리자 직접 승인 모두 포함. 이를 통해 활성 이슈 15개 이상을 상시 유지.
- **M (측정 방법):** 매일 승인 처리된 이슈 수 집계. 월간 환산 90건 이상.
- **A (달성 근거):** 현재 하루 평균 2.4건 승인 중. 이슈 자동 등록 시스템 재가동 시 3건 달성 가능.
- **R (연관성):** 이슈가 꾸준히 승인되어야 활성 이슈가 유지됨. 콘텐츠 공백 없이 회원 방문 이유를 지속 제공.
- **T (기한):** 2026년 6월 1일 이슈 자동 등록 시스템 재가동 시점부터 매일 측정

> **최종 KPI:** 2026년 6월 1일부터 이슈를 하루 최소 3건 이상 승인하여, 활성 이슈 15개 이상을 상시 유지한다. (월 환산 90건 이상)

---

### KPI 3 — 일간 숏폼 발행 건수

- **S (구체적):** YouTube Shorts·Instagram Reels·TikTok에 하루 각 1건씩, 총 3건의 숏폼이 자동으로 올라가 실제 게시 완료된 건수
- **M (측정 방법):** 각 플랫폼에 실제로 게시 완료된 숏폼 수 집계. 월간 환산 플랫폼별 25건 이상.
- **A (달성 근거):** 현재 수동 운영 성공률 50%. 자동화 안정화 후 85%+ 달성 가능. 하루 3건(플랫폼별 1건)은 자동화 완성 시 달성 가능한 수치.
- **R (연관성):** 숏폼은 직접 가입 전환보다 브랜드 인지도 역할. 하루 3건 꾸준한 발행이 알고리즘 노출 누적으로 이어져 장기적 카드뉴스 팔로워 확보에 기여.
- **T (기한):** 2026년 6월 1일 자동화 가동 시점부터 매일 측정. 매월 말일 집계.

> **최종 KPI:** 2026년 6월 1일부터 YouTube·Instagram·TikTok에 하루 각 1건씩 총 3건의 숏폼을 매일 발행한다. 월간 플랫폼별 25건 이상, 성공률 85% 이상을 달성하고 7월부터는 90% 이상을 유지한다.

| 월 | 일 목표 (총) | 월 목표 (플랫폼별) | 성공률 목표 |
|----|-----------|----------------|----------|
| 6월 | 3건/일 | 25건 | 85% |
| 7월 이후 | 3건/일 | 28건 | 90% |

### KPI 4 — 월간 투표 참여율

- **S (구체적):** 해당 월에 1회 이상 투표한 회원 수 ÷ 전체 회원 수
- **M (측정 방법):** 해당 월에 투표에 참여한 회원 수(중복 제거) ÷ 전체 회원 수
- **A (달성 근거):** 현재 지인 19명 기준 실질 참여율 약 30%. 이슈 자동 등록 시스템 가동 후 콘텐츠가 생기면 외부 회원도 유사한 비율 기대. 8월 이전은 지인 데이터라 목표 설정 의미 없음.
- **R (연관성):** 투표 = 왜난리의 핵심 액션. 투표한 회원이 재방문 가능성도 높아 서비스 건강도의 직접 지표.
- **T (기한):** 6월부터 측정 시작. 외부 유입 회원 50명 이상이 되는 8월부터 목표치 적용.

> **최종 KPI:** 2026년 8월부터 월간 투표 참여율 20% 이상을 달성하고, 연말(10~12월)에는 30% 이상을 유지한다.

| 월 | 목표 투표 참여율 |
|----|--------------|
| 6~7월 | 측정만 (기준값 수집) |
| 8월 | **20%** |
| 9월 | **25%** |
| 10~12월 | **30%** |

### KPI 5 — 월간 재방문율

- **S (구체적):** 이번 달에 방문한 회원 중 다음 달에도 방문한 비율
- **M (측정 방법):** 이번 달 방문 회원과 지난달 방문 회원을 비교해 겹치는 수 산출
- **A (달성 근거):** 6월 방문 기록 수집 코드 수정 배포 후 데이터 수집 시작. 3개월 누적 후 실제 데이터 기반으로 목표치 설정. 지금 임의로 수치를 정하면 신뢰도 없음.
- **R (연관성):** 재방문율이 낮으면 콘텐츠나 화면에 문제가 있다는 신호. 가입자 수 증가보다 서비스 품질을 더 직접적으로 반영.
- **T (기한):** 2026년 6월부터 측정 시작. 2026년 9월에 실제 데이터 기반 첫 목표치 설정.

> **최종 KPI:** 2026년 6월부터 재방문율 측정을 시작하고, 9월 데이터 확인 시점에 목표치를 확정한다. (현재 목표 수치 미설정 — 데이터 없이 설정 시 신뢰도 없음)

### KPI 6 — 월간 댓글 수

- **S (구체적):** 해당 월에 이슈 페이지에 작성된 신규 댓글 수
- **M (측정 방법):** 해당 월에 새로 작성된 댓글 수 집계
- **A (달성 근거):** 현재 기준 데이터 없음. 이슈당 평균 1~2개 기준으로 낮게 설정하고, 8월 실제 데이터 확인 후 조정.
- **R (연관성):** 댓글은 가장 적극적인 참여 형태. 댓글 작성 회원은 재방문율도 높아 커뮤니티 정착 지표로 활용.
- **T (기한):** 6월부터 측정. 8월 실제 데이터 기반으로 목표치 보정.

> **최종 KPI:** 2026년 6월부터 월간 댓글 수를 측정하고, 낮은 초기 목표를 기준점으로 삼아 8월 이후 실제 데이터로 보정한다.

| 월 | 목표 월간 댓글 수 |
|----|--------------|
| 6월 | **15개** |
| 7월 | **40개** |
| 8월 | **80개** → 실제 데이터로 보정 |
| 9월 이후 | 보정된 수치 적용 |

### KPI 7 — 월간 반응 참여율

- **S (구체적):** 해당 월에 1회 이상 반응(좋아요·화나요·팝콘각 등)을 남긴 회원 수 ÷ 전체 회원 수
- **M (측정 방법):** 해당 월에 반응을 남긴 회원 수(중복 제거) ÷ 전체 회원 수
- **A (달성 근거):** 반응은 투표보다 허들이 낮아(이모지 한 번 탭) 참여율이 더 높게 나와야 정상. 현재 지인 19명 기준 반응율 약 30%. 반응율이 투표율보다 낮으면 화면 또는 이슈 품질 문제 신호.
- **R (연관성):** 투표와 다른 행동. 반응율은 이슈에 감정적으로 공감하는 비율을 측정. 투표율과 함께 보면 회원의 참여 깊이를 입체적으로 파악 가능.
- **T (기한):** 6월부터 측정. 8월부터 목표치 적용.

> **최종 KPI:** 2026년 8월부터 월간 반응 참여율 35% 이상을 달성하고, 연말(10~12월)에는 50% 이상을 유지한다. (투표 참여율보다 항상 높게 유지되어야 정상)

| 월 | 목표 반응 참여율 |
|----|--------------|
| 6~7월 | 측정만 (기준값 수집) |
| 8월 | **35%** |
| 9월 | **42%** |
| 10~12월 | **50%** |
