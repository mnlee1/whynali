### 실제 전환 (5분 작업)

**Step 1: Claude API 키 발급 (3분)**
```
1. https://console.anthropic.com/ 접속
2. 계정 생성 또는 로그인
3. API Keys 메뉴
4. "Create Key" 클릭
5. 키 3-5개 발급 및 복사
6. 결제 정보 등록 (무료 크레딧 $5)
```

**Step 2: 환경변수 변경 (1분)**
```bash
# .env.local
AI_PROVIDER=claude  # groq → claude로만 변경

# Claude API 키 추가
ANTHROPIC_API_KEY=sk-ant-api03-키1,sk-ant-api03-키2,sk-ant-api03-키3

# Groq 키는 백업용으로 유지
GROQ_API_KEY=gsk_Za1ffV6q9s...(기존 키 유지)
```

**Step 3: Vercel 환경변수 업데이트 (1분)**
```
Vercel 대시보드:
1. Settings > Environment Variables
2. AI_PROVIDER = claude
3. ANTHROPIC_API_KEY = (발급한 키들)
4. Save
```

**Step 4: 재배포 및 확인**
```bash
git add .env.example  # 예시 파일만 커밋
git commit -m "docs: Claude API 전환 준비"
git push origin main

# Vercel 자동 재배포 (2-3분)
# 배포 후 콘솔 확인: [ClaudeProvider] 3개 API 키 로드 완료
```

**전환 소요 시간: 5분**
**전환 비용: $0** (무료 크레딧 사용)

## 📊 전환 시점 판단 가이드

### 전환하면 좋은 신호 ✅

1. **정확도 문제**
   - 카테고리 오분류 > 15%
   - 중복 이슈 > 10%
   - 뉴스 오연결 빈번

2. **비즈니스 성장**
   - 사용자 > 10,000명
   - 일일 활성 사용자 > 1,000명
   - 월간 이슈 > 120개

3. **예산 여유**
   - 월 수익 > $500
   - 투자 유치 완료
   - 예산 확보 ($150/월)

4. **사용자 불만**
   - "카테고리가 이상해요" 피드백
   - 중복 이슈 신고 증가
   - 신뢰도 하락

### 전환하지 않아도 되는 경우 ⏸️

1. **초기 단계**
   - 사용자 < 5,000명
   - 베타 테스트 중
   - 예산 제한

2. **정확도 충분**
   - 오분류율 < 10%
   - 사용자 불만 없음
   - Rate Limit 드뭄

3. **비용 우선**
   - 월 예산 < $100
   - 무료 플랜 사용 중
   - 투자 유치 전
