# 이슈 관리 페이지 전체 검증 및 수정 보고서

**작성일**: 2026-03-13  
**대상**: 왜난리 이슈 관리 시스템

## 검증 배경

사용자가 사이트에서 "윤석준 대구 동구청장 당선무효형 확정" 이슈를 발견했으나, 관리자 페이지 목록에는 표시되지 않는 문제가 발생하여 전체 이슈 데이터의 무결성을 검증하게 되었습니다.

## 발견된 문제들

### 1. source_track null 문제 (심각)

**문제**: 15개 이슈 중 11개(73.3%)가 `source_track = null`로 저장되어 있었습니다.

**원인**:
- 수동 이슈 생성 API (`/api/issues POST`)에서 `source_track` 필드를 설정하지 않음
- 관리자 페이지는 `source_track=track_a` 필터를 사용하므로 null 이슈는 표시되지 않음

**해결**:
- null 이슈 11개를 연결 데이터 기반으로 자동 분류:
  - **track_a로 재분류**: 2개 (커뮤니티 + 뉴스 연결)
  - **manual로 재분류**: 8개 (뉴스만 연결)
  - **숨김 처리**: 1개 (연결 데이터 없음)

### 2. 가짜 트랙A 이슈 (중요)

**문제**: `source_track=track_a`로 표시되었으나 실제로는 트랙A 프로세스를 거치지 않은 이슈가 4개 발견되었습니다.

**판단 기준**:
- 진짜 트랙A: 커뮤니티 글 1개 이상 + 뉴스 1개 이상 연결
- 가짜 트랙A: 커뮤니티 또는 뉴스 중 하나라도 없음

**발견된 가짜 트랙A**:
1. 윤석준 이슈 (커뮤니티 0건, 뉴스 24건)
2. 이탈리아 WBC (커뮤니티 0건, 뉴스 0건)
3. MLB 이슈 (커뮤니티 0건, 뉴스 0건)
4. WBC 한국 8강 (커뮤니티 0건, 뉴스 1건)

**해결**: 4개 모두 `manual`로 재분류 완료

### 3. 화력 미달 승인 이슈 (경미)

**문제**: 화력 0~1점(기준 15점 미달)인데 승인된 이슈 3개 발견

이들은 수동으로 승인된 것으로 보이며, 관리자 판단에 따른 것이므로 수정하지 않았습니다.

## source_track 값 정의

| 값 | 의미 | 판단 기준 |
|---|---|---|
| `track_a` | 트랙A 프로세스로 생성 | 커뮤니티 1개 이상 + 뉴스 1개 이상 연결 |
| `manual` | 수동 생성 | API를 통한 직접 생성 또는 뉴스만 연결 |
| `null` | 레거시 데이터 | **절대 발생하면 안 됨** |
