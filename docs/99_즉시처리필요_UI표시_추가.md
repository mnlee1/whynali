# 즉시 처리 필요 이슈 UI 표시 추가

> 추가일: 2026-03-11
> 파일: `app/admin/issues/page.tsx`

## 변경 사항

### 1. 이슈 목록에 시각적 표시 추가 ✅

**아이콘 추가:**
- ⚠️ 주황색 경고 아이콘 (제목 왼쪽)
- 툴팁: "즉시 처리 필요 (화력 30점 이상)"

**뱃지 추가:**
- "즉시 처리 필요" 빨간색 뱃지
- 배경: `bg-red-100`, 텍스트: `text-red-700`

**표시 조건:**
```typescript
issue.approval_status === '대기' && 
(issue.heat_index ?? 0) >= 30 && 
['연예', '정치'].includes(issue.category)
```

### 2. 가이드 섹션에 설명 추가 ✅

**새 섹션 "⚠️ 즉시 처리 필요" 추가:**
- 화력 30점 이상 + 연예/정치
- 승인 상태: 대기
- UI 표시: ⚠️ 아이콘 + 뱃지
- Dooray 알림 발송

**자동 승인 조건 업데이트:**
- 연예 카테고리 자동 승인 가능으로 변경
- 정치만 관리자 승인 필수로 명시

## UI 구성

### 이슈 목록 표시 순서

```
[아이콘들] 제목 [뱃지들]
```

**예시 1: 급증 이슈**
```
🔥 "OO 사고 관련 논란" [뉴스 급증]
```

**예시 2: 즉시 처리 필요 (급증 아님)**
```
⚠️ "국힘 尹 정치적 복귀 반대" [즉시 처리 필요]
```

**예시 3: 급증이면서 즉시 처리 필요**
```
🔥 ⚠️ "OO 정치인 긴급 발언" [뉴스 급증] [즉시 처리 필요]
```

**예시 4: 일반 이슈**
```
"일반 이슈 제목"
```

## 아이콘/뱃지 우선순위

### 아이콘 (제목 왼쪽)
1. 🔥 급증 이슈 (`is_urgent = true`)
2. ⚠️ 즉시 처리 필요 (급증 아닌 경우만)

### 뱃지 (제목 오른쪽)
1. "커뮤니티 급증" (보라색)
2. "뉴스 급증" (주황색)
3. "즉시 처리 필요" (빨간색)

## 코드 구조

```typescript
<div className="flex items-center gap-2">
    {/* 급증 아이콘 */}
    {issue.is_urgent && (
        <span className="text-red-500" title="급증 이슈">🔥</span>
    )}
    
    {/* 즉시 처리 필요 아이콘 (급증 아닌 경우만) */}
    {!issue.is_urgent && 
     issue.approval_status === '대기' && 
     (issue.heat_index ?? 0) >= 30 && 
     ['연예', '정치'].includes(issue.category) && (
        <span className="text-orange-500" title="즉시 처리 필요 (화력 30점 이상)">⚠️</span>
    )}
    
    {/* 제목 링크 */}
    <a href={`/issue/${issue.id}`} target="_blank">
        {issue.title}
    </a>
    
    {/* 급증 뱃지들 */}
    {issue.source_track === 'community_burst' && (
        <span className="...">커뮤니티 급증</span>
    )}
    {issue.source_track === 'news_collection' && issue.is_urgent && (
        <span className="...">뉴스 급증</span>
    )}
    
    {/* 즉시 처리 필요 뱃지 */}
    {issue.approval_status === '대기' && 
     (issue.heat_index ?? 0) >= 30 && 
     ['연예', '정치'].includes(issue.category) && (
        <span className="...">즉시 처리 필요</span>
    )}
</div>
```

## 효과

### 변경 전
- 화력 65점 정치 이슈가 일반 대기 이슈와 구분 안 됨
- 관리자가 화력 수치를 일일이 확인해야 함
- 긴급 처리 대상 파악 어려움

### 변경 후
- ⚠️ 아이콘으로 즉시 시각적으로 구분
- "즉시 처리 필요" 뱃지로 명확한 상태 표시
- Dooray 알림과 UI 표시가 일치하여 일관성 확보
- 관리자 업무 효율 향상

## 연관 기능

### Dooray 알림
- **즉시 알림**: 이슈 생성 시점에 메신저로 전송
- **배치 알림**: 매시 정각에 대기 중인 목록 전송
- 알림 조건 = UI 표시 조건 (동일)

### 자동 승인
- 연예: 화력 30점 이상이면 자동 승인 (Dooray 알림 발송)
- 정치: 화력과 무관하게 관리자 승인 필수
- 즉시 처리 필요 = 정치 고화력 대기 이슈

- 표시: 🔥 + "뉴스 급증" + "즉시 처리 필요"
- 복합 케이스
