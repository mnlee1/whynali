### 성능 확인

```sql
-- 이슈별 연결 뉴스 수
SELECT 
    i.title,
    i.category,
    COUNT(n.id) as news_count
FROM issues i
LEFT JOIN news_data n ON n.issue_id = i.id
WHERE i.created_at > NOW() - INTERVAL '1 day'
GROUP BY i.id, i.title, i.category
ORDER BY news_count DESC;
```
