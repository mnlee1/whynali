## 📊 Claude 3.5 Sonnet 현재 가격

### 공식 가격표 (2026년 기준)

**Claude 3.5 Sonnet (claude-3-5-sonnet-20241022):**
- Input: **$3.00 / 1M tokens**
- Output: **$15.00 / 1M tokens**

### 왜난리 프로젝트 비용 계산

**일일 사용량:** 150,000 tokens
- Input: 105,000 tokens (70%)
- Output: 45,000 tokens (30%)

**일일 비용:**
```
Input:  105,000 × ($3.00 / 1M) = $0.315
Output: 45,000 × ($15.00 / 1M) = $0.675
총:     $0.99/일
```

**월간 비용:**
```
$0.99 × 30일 = $29.70/월 (약 38,610원)
```
