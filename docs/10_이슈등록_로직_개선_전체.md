# 이슈 등록 로직 개선 전체 히스토리

날짜: 2026-02-26

## 1. 요구사항 분석

### 기존 문제점

1. 제목 기반 매칭의 한계: 키워드 토큰화만으로는 같은 이슈를 정확히 묶기 어려움
2. 무분별한 자동 승인: 모든 카테고리가 동일 기준으로 자동 승인되어 연예/정치 같은 민감 분야에서 오등록 위험
3. 재점화 미감지: 종결된 이슈에 대한 신규 데이터 급증 시 자동 상태 전환 없음

### 개선 방안

1. 조건부 자동 승인: 카테고리별 차별화된 승인 정책
2. 재점화 감지: 급증/점진적 재점화 자동 감지
3. 태그 기반 매칭: AI 태그 추출로 그루핑 정확도 향상 (보류)

## 2. 구현 완료 항목

### 2.1 조건부 자동 승인

목적: 카테고리별 민감도에 따라 자동 승인 조건 차별화

환경변수:
```bash
AUTO_APPROVE_CATEGORIES=사회,기술,스포츠
```

카테고리별 정책:

| 카테고리 | 자동 승인 | 이유 |
|---------|---------|------|
| 사회 | 허용 | 객관적 사건/사고, 리스크 낮음 |
| 기술 | 허용 | 제품 출시, 업계 동향 등 사실 중심 |
| 스포츠 | 허용 | 경기 결과, 이적 등 명확한 사실 |
| 연예 | 금지 | 루머, 프라이버시 민감 |
| 정치 | 금지 | 정치적 편향 리스크 |

### 2.2 재점화 감지

목적: 종결된 이슈의 신규 데이터 급증 시 자동으로 '논란중' 복귀

환경변수:
```bash
STATUS_REIGNITE_RATE_PER_MINUTE=5
STATUS_REIGNITE_DURATION_MINUTES=10
```

동작:
1. 급증 감지 (최우선): 10분간 50건 이상 → 즉시 논란중
2. 점진적 재점화: 48시간 내 신규 수집 + 화력 40점 이상 → 논란중

### 2.3 태그 기반 매칭 (보류)

목적: AI 추출 태그로 그루핑 정확도 향상

보류 이유:
1. Perplexity AI 품질 문제
2. 현재 핵심 인물 리스트로 충분한 정확도
3. 추가 비용 발생 (월 4원)

향후 검토: OpenAI API 사용 시 재검토

## 3. 테스트 결과

### 3.1 자동 승인 검증

- 19개 이슈 자동 승인 (카테고리 기반)
- 4개 저화력 이슈 자동 반려
- 자동 승인률: 68% (19/28)

### 3.2 재점화 감지

- 로직 구현 완료
- 실제 종결 이슈 발생 시 작동 예정

### 3.3 저품질 이슈 정리

문제:
- 임계값 테스트 시 MIN_HEAT_TO_REGISTER=10으로 낮춰서 저품질 이슈 25개 등록됨
- 화력 7~19점 범위의 이슈들이 공개 목록에 노출

조치:
- 25개 저품질 이슈 반려 처리 완료
- 최종 공개 이슈: 3개 (모두 화력 30점 이상)
  - 옥택연 결혼 (화력 58점, 연예)
  - 민희진 뉴진스 (화력 30점, 연예)
  - 갤럭시 S26 (화력 55점, 기술)

### 3.4 그루핑 오류 개선

문제 사례:
```
A: '민주당 갈라치기 안 먹혔다' 조국당 지지율 최저치 경신 (한국 정치)
B: 미 민주당 반박 연설 "트럼프, 전례 없는 부패 대통령" (미국 정치)
→ "민주당" 키워드만 겹쳐서 완전히 다른 주제가 같은 이슈로 묶임
```

해결책: 불용어 확장
- 정당/정파: 민주당, 공화당, 여당, 야당, 보수, 진보 등
- 국가/지역: 북한, 러시아, 영국, 프랑스, 유럽, 아시아 등
- 범용 정치 용어: 발언, 언급, 입장, 반박, 대응, 논평 등

개선 효과:
- 오묶임 비율: 15% → 3% (예상)
- 이슈 정확도: 70% → 90% (예상)

### 3.5 제품명 매칭 오류 개선

문제 사례:
```
커뮤니티: "색 잘뽑은거 같은 갤럭시 버즈4 핑크 골드.JPG"
이슈: "멀티 AI 품은 갤럭시S26 공개 D-1"
→ "갤럭시" 브랜드명만 겹쳐서 완전히 다른 제품(버즈4 vs S26)이 매칭됨
```

해결책:
1. 커뮤니티 매칭 임계값 상향: 1 → 2
2. 제품명 특수 처리: 브랜드 + 모델명 조합은 완전 일치만 허용

### 3.6 커뮤니티 오매칭 개선

문제 사례:
```
이슈: "짐승남 옥택연, 품절남 된다 '4월의 신랑'"
뉴스: "옥택연, 10년 사귄 연인과 4월 24일 결혼"

잘못 매칭된 커뮤니티:
1. "한국에서 활동하는 외국인들에게도 적용된다는 아이돌 세대 법칙"
2. "한국영화계를 위해 왕사남이 흥행하면 안된다는 자칭 시네필들"
3. "멈추지않는지랄이계속된다"
4. "아이브 가을이가 명품엠버서더에 발탁된다면?"
```

원인: "된다", "활동", "발탁" 같은 범용 동사가 불용어가 아님

해결책:
1. 범용 동사/조사 불용어 추가: 있다, 없다, 한다, 된다, 이다, 같다 등
2. 핵심 키워드 로직 개선: 3글자 이상 + 범용 단어 제외

검증:
```
옥택연 이슈:
Before: 4건 (전혀 관련 없는 커뮤니티)
After: 0건 (개선된 로직으로 모두 제거됨)
```

## 5. 기대 효과

### 조건부 자동 승인
- 연예/정치 이슈의 오등록 리스크 감소
- 사회/기술/스포츠 이슈의 빠른 등록
- 관리자 부담 감소 (민감 분야만 검토)

### 재점화 감지
- 재점화 이슈 자동 감지 (급증 + 점진적)
- 사용자 경험 개선 (빠른 상태 반영)
- 관리자 수동 작업 감소

### 그루핑/매칭 개선
- 오묶임/오매칭 대폭 감소
- 이슈 정확도 향상
- 품질 높은 이슈만 등록
