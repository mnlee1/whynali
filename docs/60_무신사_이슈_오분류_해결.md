# 무신사 이슈 카테고리 오분류 해결

날짜: 2026-03-11
상태: ✅ 완료

## 문제 상황

"무신사, 스포츠·IP 전문 큐레이션관 '팬 스토어' 론칭…'팬덤 경제' 공" 이슈가 잘못 분류되고 무관한 뉴스가 연결된 문제 발견.

### 발견된 문제

```
이슈 제목: 무신사, 스포츠·IP 전문 큐레이션관 '팬 스토어' 론칭…'팬덤 경제' 공
이슈 ID: eda4c92d-d0ba-45c3-a068-b5a4ce40ea79

잘못된 분류:
├─ 카테고리: 스포츠 ❌
└─ 연결된 뉴스: 95건 (모두 무관한 스포츠 뉴스)

올바른 분류:
├─ 카테고리: 기술 ✅ (이커머스 플랫폼의 신규 서비스)
└─ 연결 뉴스: 10건 (무신사 팬 스토어 관련 뉴스)
```

### 잘못 연결되었던 뉴스 예시

```
- 문체부, 54개 '어르신 스포츠 지원사업' 선정
- 스포츠웨어 기업 '언더아머', 개인정보 유출
- 방송협회 "스포츠 중계권 독점 해결하라"
- 인천대, 스포츠산업 예비·초기창업기업 모집
- 용인시, 생활체육 AI 스포츠 중계 시스템 도입
...
```

## 원인 분석

### 1. 카테고리 분류 오류

**키워드 기반 분류의 한계:**
```typescript
제목: "무신사, 스포츠·IP 전문 큐레이션관..."
키워드: "스포츠" 포함
→ 스포츠 카테고리로 잘못 분류
```

**맥락 이해 부족:**
- "스포츠·IP"는 스포츠 굿즈/브랜드 상품 카테고리를 의미
- 실제 스포츠 경기/선수와는 무관한 **유통/이커머스 서비스**
- 무신사는 패션 이커머스 플랫폼 → **기술** 카테고리가 적합

### 2. 뉴스 링킹 오류

**카테고리 필터 미작동:**
```typescript
// lib/linker/issue-news-linker.ts
// 이슈가 스포츠로 분류되어, 스포츠 뉴스만 검색 대상
.or(`category.eq.${issueCategory},category.is.null`)
```

**"스포츠" 키워드 오매칭:**
- 무신사 이슈: "스포츠·IP 큐레이션"
- 무관한 뉴스: "어르신 스포츠 지원", "스포츠 중계권"
- 공통 키워드: "스포츠" 1개만 일치 → 임계값 미달이어야 하나 카테고리 동일로 인해 매칭됨

### 1. 카테고리 키워드 개선

**기술 카테고리에 이커머스 키워드 추가:**
```typescript

    '패션', '브랜드', '큐레이션', '스토어', '론칭', '입점', '판매',

```

**Context Rules 추가:**

### 3. AI 프롬프트 개선

**명확한 구분 기준 추가:**
```typescript
// lib/candidate/category-classifier.ts
주의사항:
- "스포츠마케팅"처럼 스포츠 관련 단어가 있어도, 주제가 지역 관광/마케팅이면 "사회"
- "스포츠·IP 큐레이션", "스포츠브랜드" 등은 유통/패션 서비스이므로 "기술"
- 무신사, 쿠팡, 네이버쇼핑 등 이커머스 플랫폼의 신규 서비스는 "기술"
- 실제 경기 결과, 선수 활동, 대회 관련만 "스포츠"
```

### 3. 모니터링 강화

**이상 패턴 탐지:**
```sql
-- 카테고리별 이슈 수 모니터링
SELECT category, COUNT(*) as count
FROM issues
WHERE approval_status = '승인'
GROUP BY category
ORDER BY count DESC;

-- 연결 뉴스 수가 비정상적으로 많은 이슈
SELECT id, title, category, 
    (SELECT COUNT(*) FROM news_data WHERE issue_id = issues.id) as news_count
FROM issues
WHERE approval_status = '승인'
HAVING news_count > 50
ORDER BY news_count DESC;
```
