## 1. 즉시 수정 필요 사항

### 1.1 오픈ID 로그인 미완성 (중요도: 높음)

**현황**
- 네이버 로그인만 활성화
- 구글, 카카오 OAuth 키 미설정

**위치**
- `app/login/page.tsx` 89-106줄
- `.env.local` 14-19줄

**필요 작업**
1. 구글 OAuth 설정
   - Google Cloud Console 앱 등록
   - Client ID, Client Secret 발급
   - `.env.local`에 `NEXT_PUBLIC_GOOGLE_CLIENT_ID`, `GOOGLE_CLIENT_SECRET` 설정

2. 카카오 OAuth 설정
   - Kakao Developers 앱 등록
   - REST API 키, Secret 발급
   - `.env.local`에 `NEXT_PUBLIC_KAKAO_CLIENT_ID`, `KAKAO_CLIENT_SECRET` 설정

3. 로그인 페이지에서 버튼 활성화
   - 현재 주석 처리된 구글, 카카오 로그인 버튼 활성화

**기획 요구사항**
- 02_AI기획_판단포인트.md §3: 구글·네이버·카카오 3종 필수 포함

## 2. UI/UX 개선 권장 사항

### 2.1 이슈 목록 정렬 옵션 미흡 (중요도: 중간)

**현황**
- API는 화력순 정렬 지원 (`sort=heat`)
- UI에 화력순 정렬 옵션 없음
- 기본 정렬이 최신순만 적용

**기획 요구사항**
- 02_AI기획_판단포인트.md §4: 복합 정렬 (상태+최신+화력)
- 07_이슈등록_화력_정렬_규격.md §3.3: 기본 정렬은 `heat`, 선택 옵션으로 `latest`

**필요 작업**
1. 이슈 목록에 정렬 드롭다운 또는 탭 추가
   - 최신순
   - 화력순
   - 복합 정렬 (상태 우선 → 최신순 → 화력순)

2. 복합 정렬 구현
   - 1차: 현재 상태 (점화/논란중 우선, 종결 하단)
   - 2차: 최신성 (updated_at DESC)
   - 3차: 화력 (heat_index DESC)

**위치**
- `components/issues/IssueList.tsx`
- `app/api/issues/route.ts`

## 4. 환경 변수 누락 사항

### 4.1 소셜 로그인 키 누락

**.env.local 확인 결과**

```
# 설정됨
NAVER_CLIENT_ID=3R2LcyDFmcC58BKqUNHI
NAVER_CLIENT_SECRET=YsRvAIHP75

# 누락됨
NEXT_PUBLIC_KAKAO_CLIENT_ID=
KAKAO_CLIENT_SECRET=
NEXT_PUBLIC_GOOGLE_CLIENT_ID=
GOOGLE_CLIENT_SECRET=
```

**필요 작업**
1. 카카오 Developers에서 앱 등록 후 키 발급
2. Google Cloud Console에서 OAuth 설정 후 키 발급
3. `.env.local` 및 Vercel 환경 변수에 등록

---

### 4.2 선택 API 키 (기능별 필요)

**현재 설정됨**
- `GROQ_API_KEY`: AI 토론 주제/투표 생성 (다중 키 설정됨)
- `AI_PROVIDER=groq`: AI 프로바이더 선택

**누락 (선택 사항)**
- `OPENAI_API_KEY`: AI 투표 생성 우선 사용 (없으면 Groq 사용)
- `PERPLEXITY_API_KEY`: AI 그루핑/매칭 (ENABLE_AI_GROUPING=true 시)
- `RESEND_API_KEY`: 이메일 알림 (알림 기능 구현 시)
