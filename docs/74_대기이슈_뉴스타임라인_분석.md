## 근본 원인 정리

### A. 뉴스 미연결 원인

#### 1. 키워드 매칭 실패 (70% 임계값)
```typescript
// lib/linker/issue-news-linker.ts
const threshold = Math.min(5, Math.max(3, Math.ceil(keywordsLower.length * 0.7)))
```

**문제점:**
- 이슈 제목의 키워드와 뉴스 제목의 단어가 70% 일치해야 함
- 전문 용어나 고유명사가 많으면 매칭 실패 확률 높음
- 예: "UNIST", "폐PET", "사회보장정보원" 등

#### 2. 카테고리 불일치
```typescript
// 이슈 카테고리와 뉴스 카테고리가 동일해야 연결됨
.or(`category.eq.${issueCategory},category.is.null`)
```

**문제점:**
- AI 카테고리 분류가 부정확할 수 있음
- "유한재단 돌봄 사업" → 기술(X), 사회(O)
- 카테고리 오분류 시 관련 뉴스 전혀 연결 안 됨

#### 3. 날짜 범위 제한
```env
LINKER_NEWS_BEFORE_DAYS=30  # 이슈 생성 전 30일
LINKER_NEWS_AFTER_DAYS=30   # 이슈 생성 후 30일
```

**문제점:**
- 이슈가 뉴스보다 늦게 생성되면 매칭 어려움
- 예: 오늘 생성된 이슈 + 2개월 전 보도 → 미연결

#### 4. AI 검증 실패
```typescript
// AI가 관련도 70% 미만으로 판단하면 연결 안 됨
if (result.isRelated && result.confidence >= 70)
```

**문제점:**
- 키워드 매칭은 통과했지만 AI가 거부
- 너무 보수적인 판단 가능성

### B. 타임라인 미생성 원인

#### 1. Cron 실행 주기 문제
```yaml
# .github/workflows/cron-auto-timeline.yml
schedule:
  - cron: '*/30 * * * *'  # 30분마다
```

**문제점:**
- 뉴스 연결 후 최대 30분 대기
- 최근 생성된 이슈는 다음 Cron까지 대기

#### 2. 조건 미충족
```typescript
// lib/timeline/auto-timeline.ts
if ((count ?? 0) > 0) return 0  // 이미 타임라인 있으면 skip

if (!news || news.length === 0) return 0  // 뉴스 없으면 skip
```

**문제점:**
- 뉴스가 없으면 타임라인 생성 불가
- 뉴스 미연결 → 타임라인 미생성 연쇄 효과

**효과:**

#### 2. 카테고리 수정
```

  현재: 기술 ❌
  수정: 사회 ✅
```

**방법:**
1. 관리자 페이지 → 이슈 관리
2. 해당 이슈 선택 → 카테고리 변경
3. auto-link Cron 재실행

**효과:**
- 뉴스 미연결 1건 해결

#### 3. 이슈 제목 개선
```
UNIST 폐PET 분해 기술:

```

**효과:**
- 핵심 키워드 중심으로 단순화
- 매칭 확률 증가

### 시스템 개선 (중기)

#### 1. 날짜 범위 확대
```env

```

**효과:**

**트레이드오프:**
- 무관한 뉴스 연결 위험 증가
- AI 검증으로 보완 필요

#### 2. 키워드 임계값 조정
```typescript
// lib/linker/issue-news-linker.ts

// 개선: 60% (0.6)

```

**효과:**
- 더 많은 뉴스 매칭
- 재현율(Recall) 증가

**트레이드오프:**
- 오연결 위험 증가
- AI 검증 필수

#### 3. 카테고리 교차 검색
```typescript
// 현재: 동일 카테고리만

// 개선: 인접 카테고리 포함

    '사회': ['사회', '기술'],
    // ...
}
```

**효과:**
- 카테고리 경계 케이스 해결
- 오분류 영향 감소

#### 4. AI 검증 신뢰도 조정
```typescript

// 개선: 60% 이상

```

**효과:**
- 더 유연한 연결
- 재현율 증가

**트레이드오프:**
- 정확도 약간 감소 (95% → 90%)
